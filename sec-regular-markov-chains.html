    Section 15.4 Regular Markov chains

    When does \(P^k\vect{x} = \vect{x}_k\) tend towards a unique steady state vector? The next example shows that steady state vectors do not need to be unique.

    Example 15.4.1.

    Consider a Markov chain for a population of 600 people, with transition probabilities given by FigureÂ 15.4.2.
    ðŸ”—
    First, the transition matrix is
    \begin{equation*} P = \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1\\ 0 \amp 1 \amp 0 \end{bmatrix}\text{.} \end{equation*}
    Solving \((P-I)\vect{x} = \vect{0}\text{,}\) we get
    \begin{equation*} \begin{amatrix}{c} P - I \amp 0 \end{amatrix} = \begin{amatrix}{ccc} 0 \amp 0 \amp 0 \amp 0\\ 0 \amp -1 \amp 1 \amp 0\\ 0 \amp 1 \amp -1 \amp 0 \end{amatrix} \xrightarrow{R_3 \mapsto R_3 + R_2} \begin{amatrix}{ccc} 0 \amp 0 \amp 0 \amp 0\\ 0 \amp -1 \amp 1 \amp 0\\ 0 \amp 0 \amp 0 \amp 0 \end{amatrix}\text{.} \end{equation*}
    Let \(x = s, z = t\text{.}\) Then the second row gives \(y=z=t\text{.}\) So the solutions are
    \begin{equation*} \left\{s \begin{bmatrix} 1 \\ 0 \\ 0 \end{bmatrix} + t \begin{bmatrix} 0 \\ 1\\ 1 \end{bmatrix} \;\middle| \; s, t \in \R\right\}\text{.} \end{equation*}
    So there are two different steady state vectors, \([600, 0, 0]\) (where \(s = 600\text{,}\) \(t=0\)) and \([0, 300, 300]\) (where \(s = 0\) and \(t = 300\)). There are also two different SSPVs \([1, 0, 0]\) (from \(s=1\text{,}\) \(t=0\)) and \([0, 1/2, 1/2]\) (from \(s=0, t=1/2)\text{.}\)
    ðŸ”—
    Notice that if the initial state vector is
    \begin{equation*} \vect{x}_0 = \begin{bmatrix} 600 \\ 0 \\ 0 \end{bmatrix} , \qquad \text{then }\, \vect{x}_k = \begin{bmatrix} 600 \\ 0 \\ 0 \end{bmatrix} \, \text{for all } k\ge 1, \end{equation*}
    because \(\vect{x}_0\) is a SSV. But if the initial state vector is \(\vect{x}_0 = [0, 300, 300]\text{,}\) the other SSV, then \(\vect{x}_k = [0, 300, 300]\) for all \(k\ge 1\) instead.
    ðŸ”—

    Example 15.4.5.

    A psychologist places a rat in a cage as in FigureÂ 15.4.6.
    ðŸ”—
    The rat has been trained so that each time it hears a bell, it changes compartment, and it is equally to choose each door. Describe the long term behaviour of the rat.
    ðŸ”—
    Based on the number of doors, the transition matrix is
    \begin{equation*} P = \begin{bmatrix} 0 \amp 1/3 \amp 1/3 \\ 1/2 \amp 0 \amp 2/3 \\ 1/2 \amp 2/3 \amp 0 \end{bmatrix}\text{.} \end{equation*}
    Since \(P^2\) is
    \begin{equation*} \begin{bmatrix} 1/3 \amp 2/9 \amp 2/9 \\ 1/3 \amp 11/18 \amp 1/6 \\ 1/3 \amp 1/6 \amp 11/18 \end{bmatrix} \end{equation*}
    which is positive, \(P\) is regular. Therefore TheoremÂ 15.4.4 applies. Solving for the unique steady-state probability vector:
    \begin{align*} \begin{amatrix}{c} P - I \amp \vect{0} \end{amatrix} = \begin{amatrix}{ccc} -1 \amp 1/3 \amp 1/3 \amp 0\\ 1/2 \amp -1 \amp 2/3 \amp 0\\ 1/2 \amp 2/3 \amp -1 \amp 0 \end{amatrix} \amp\xrightarrow{\text{scaling}} \begin{amatrix}{ccc} -3 \amp 1 \amp 1 \amp 0\\ 3 \amp -6 \amp 4 \amp 0\\ 3 \amp 4 \amp -6 \amp 0 \end{amatrix} \\ \amp\xrightarrow[R_3 \mapsto R_3 + R_1]{R_2 \mapsto R_2 + R_1} \begin{amatrix}{ccc} -3 \amp 1 \amp 1 \amp 0\\ 0 \amp -5 \amp 5 \amp 0\\ 0 \amp 5 \amp -5 \amp 0 \end{amatrix} \\ \amp\xrightarrow{R_3 \mapsto R_3 + R_2} \begin{amatrix}{ccc} -3 \amp 1 \amp 1 \amp 0\\ 0 \amp -5 \amp 5 \amp 0\\ 0 \amp 0 \amp 0 \amp 0 \end{amatrix}\text{.} \end{align*}
    Let \(z = t\text{.}\) Then \(-5y + 5z = 0\text{,}\) so \(y=z=t\text{,}\) and \(-3x+y+z=0\) implies that \(x = 2t/3\text{.}\) So
    \begin{equation*} E_1(P) = \left\{t \begin{bmatrix} 2/3 \\ 1\\ 1 \end{bmatrix} : t \in \R\right\}\text{.} \end{equation*}
    ðŸ”—
    For the SSPV we need \(2t/3 + t + t = 8t/3 = 1\text{,}\) so \(t = 3/8\text{.}\) Therefore the SSPV is
    \begin{equation*} \vect{x} = \begin{bmatrix} 1/4 \\ 3/8\\ 3/8 \end{bmatrix}\text{.} \end{equation*}
    This means that in the long term, the rat is in compartment 1 with probability 1/4, compartment 2 with probability 3/8, and compartment 3 with probability 3/8.
