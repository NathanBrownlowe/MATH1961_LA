    Section 5.3 Number of solutions

    In most of the examples so far, and all of the examples which we have solved via Gaussian elimination, the system has had a single solution. In fact, this is only one of three possibilities.

    Theorem 5.3.1.

    Every system of linear equations has either
    1. a unique solution
      ðŸ”—
      ðŸ”—
    2. infinitely many solutions
      ðŸ”—
      ðŸ”—
    3. no solutions.
      ðŸ”—
      ðŸ”—
    ðŸ”—
    ðŸ”—
    In other words, it is impossible for the number of solutions to be 2, 3, \dots or any finite number greater than 1.
    ðŸ”—

    Definition 5.3.2.

    A system of linear equations is called consistent if it has at least one solution, and inconsistent if it has no solutions.
    ðŸ”—
    ðŸ”—
    We will not prove TheoremÂ 5.3.1, but we will now sketch an argument in two special geometric cases, so that hopefully it seems reasonable.
    ðŸ”—
    Firstly, if we have two equations in two variables
    \begin{equation*} \begin{cases} a_1x +b_1y \amp = c_1 \\ a_2x + b_2y \amp = c_2 \end{cases}, \end{equation*}
    then each equation is a line in \(\R^2\text{,}\) so the solutions describe the intersection of two lines in \(\R^2\text{.}\) The only possibilities are one intersection, if the lines are not parallel; no solutions, if the lines are parallel and do not coincide; and infinitely many solutions if the lines coincide. Actually, we have already seen this in ExampleÂ 5.1.5 to ExampleÂ 5.1.7, and we briefly discussed this there.
    ðŸ”—
    Secondly, if we have three equations in three variables, each equation is the general form of a plane in \(\R^3\text{,}\) so the number of solutions is the number of intersections of these planes. There can be a unique solution if all 3 planes meet at a single point, infinitely many solutions if the planes intersect along a line, or no solutions when the planes have no points in common. These possibilities are illustrated in FigureÂ 5.3.3 and FigureÂ 5.3.4. A similar geometric argument works for \(n\) equations in \(n\) variables, though it becomes harder to visualise.
    ðŸ”—
    Gaussian elimination for equations with no or infinitely many solutions: The first step is always to get the augmented matrix into REF. Once youâ€™ve done this, hereâ€™s how to tell how many solutions there are, and what to do next.
    • If in REF we have a row of the form
      \begin{equation*} \begin{amatrix}{cccc} 0 \amp 0 \amp \dots \amp 0 \amp b \end{amatrix} \end{equation*}
      where \(b\) is nonzero, then we conclude that there are no solutions. This is because it is impossible for \(0x_1 + 0x_2 + \dots + 0x_n\) to equal something nonzero.
      ðŸ”—
      ðŸ”—
    • Look for any columns such that all its entries are not the leading entries of any row. (This can occur if there are less equations than the number of variables, or if there is a zero row in REF.) If there are any, then you will need to assign these variables to a parameter. For example, consider the REF augmented matrix
      \begin{equation*} \begin{amatrix}{ccc} 1 \amp 8 \amp 0 \amp 1\\ 0 \amp 2 \amp 1 \amp 2\\ \end{amatrix} \end{equation*}
      Then the \(z\)-column doesnâ€™t contain any leading entries, so we introduce the parameter \(t \in \R\text{,}\) and let \(z=t\text{.}\) Now solve for the rest of the variables in terms of the parameter. There will be infinitely many solutions (one for each value of the parameter).
      ðŸ”—
      ðŸ”—
    • If neither of the two above situations apply, then there is a unique solution, and we can solve for it as usual.

    Example 5.3.5.

    Show that the system of linear equations
    \begin{equation*} \begin{cases} x + 4y -z \amp = 0 \\ x + 6y \amp = 3 \\ 2x + 10y - z \amp = 5 \end{cases}\text{.} \end{equation*}
    has no solutions.
    ðŸ”—
    Row reducing gives
    \begin{align*} \begin{amatrix}{ccc} 1 \amp 4 \amp -1 \amp 0\\ 1 \amp 6 \amp 0 \amp 3\\ 2 \amp 10 \amp -1 \amp 5\\ \end{amatrix} \amp\xrightarrow[R_3\mapsto R_3 - 2R_1]{R_2 \mapsto R_2 - R_1} \begin{amatrix}{ccc} 1 \amp 4 \amp -1 \amp 0\\ 0 \amp 2 \amp 1 \amp 3\\ 0 \amp 2 \amp 1 \amp 5 \end{amatrix} \\ \amp\xrightarrow{R_3 \mapsto R_3 - R_2} \begin{amatrix}{ccc} 1 \amp 4 \amp -1 \amp 0\\ 0 \amp 2 \amp 1 \amp 3\\ 0 \amp 0 \amp 0 \amp 2 \end{amatrix}\text{.} \end{align*}
    The last row is a zero row with 2 after the line, so there is no solution.
    ðŸ”—
    ðŸ”—
    The next example shows how to solve linear equations with infinitely many solutions, by introducing a parameter. In this example, we will only need to introduce one parameter. Sometimes it is necessary to introduce multiple parameters; this will be covered later in ExampleÂ 5.4.6.
    ðŸ”—

    Example 5.3.6.

    Solve the system of linear equations
    \begin{equation*} \begin{cases} 2x + y +5z \amp = 1 \\ 2x + 3y +4z \amp = 4 \\ 4x + 11 z \amp = -1 \end{cases}\text{.} \end{equation*}
    has no solutions.
    ðŸ”—
    Row reducing gives
    \begin{equation*} \begin{amatrix}{ccc} 2 \amp 1 \amp 5 \amp 1\\ 2 \amp 3 \amp 4 \amp 4\\ 4 \amp 0 \amp 11 \amp -1\\ \end{amatrix} \to \begin{amatrix}{ccc} 2 \amp 1 \amp 5 \amp 1\\ 0 \amp 2 \amp -1 \amp 3\\ 0 \amp -2 \amp 1 \amp -3 \end{amatrix} \to \begin{amatrix}{ccc} 2 \amp 1 \amp 5 \amp 1\\ 0 \amp 2 \amp -1 \amp 3\\ 0 \amp 0 \amp 0 \amp 0 \end{amatrix}\text{.} \end{equation*}
    The last column has no leading entries, so \(z\) is not determined. However, this is fine: we let \(z = t\text{,}\) for \(t \in \R\) a parameter. Now we can back-substitute in terms of \(t\text{.}\) We have \(2y - z = 3\text{,}\) so \(y = \frac{1}{2}(3+z) = \frac{3}{2} + \frac{1}{2}t\text{.}\) Finally,
    \begin{align*} 2x + y +5z = 1 \implies x \amp = \frac{1}{2}(1-y - 5z) \\ \amp = \frac{1}{2}\Bigl(1 - \Bigl(\frac{3}{2}+\frac{1}{2}t\Bigr) -5t\Bigr) \\ \amp = -\frac{1}{4}-\frac{11}{4}t\text{.} \end{align*}
    So there are infinitely many solutions \([x, y, z] = [-\frac{1}{4}-\frac{11}{4}t, \frac{3}{2} + \frac{1}{2}t, t]\text{,}\) \(t \in \R\text{.}\)
    ðŸ”—
    ðŸ”—

    \begin{equation*} \vect{x} = \Bigl[-\frac{1}{4}, \frac{3}{2}, 0\Bigr] + t\Bigl[-\frac{11}{4}, \frac{1}{2}, 1\Bigr], \qquad t\in \R\text{.} \end{equation*}
    Notice that to save space and time, we didnâ€™t label the arrows during row reduction. We will sometimes do the same thing in future examples.
