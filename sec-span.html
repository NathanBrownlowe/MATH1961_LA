    Definition 8.1.1.

    Let \(S = \{\vect{v}_1, \vect{v}_2, \dots, \vect{v}_k\}\) be a set of vectors in \(\R^n\text{.}\) The span of \(S\text{,}\) or the span of \(\vect{v}_1, \vect{v}_2, \dots, \vect{v}_k\text{,}\) is the set of all linear combinations of \(\vect{v}_1, \vect{v}_2, \dots, \vect{v}_k\text{,}\) and is denoted by \(\lspan{(S)}\) or \(\lspan{(\vect{v}_1, \vect{v}_2, \dots, \vect{v}_k)}\text{.}\)

    Example 8.1.5.

    Is \([8, -5, -15] \in \lspan{([1, 2, 0], [-2, 3, 5]})\text{?}\)
    ðŸ”—
    We need to look for \(c_1, c_2 \in\R\) such that
    \begin{equation*} [8, -5, -15] = c_1[1, 2, 0] +c_2[-2, 3, 5]\text{.} \end{equation*}
    Comparing components gives the system of linear equations
    \begin{equation*} \begin{cases} c_1 - 2c_2 \amp = 8 \\ 2c_1 + 3c_2 \amp = -5 \\ 5c_2 \amp = -15 \end{cases}\text{.} \end{equation*}
    The third equation gives that \(c_2 = -3\text{,}\) and then the first equation gives that
    \begin{equation*} c_1 = 8+ 2c_2 = 8 + 2(-3) = 2\text{.} \end{equation*}
    These values also satisfy the second equation, since \(2(2) + 3(-3) = -5\text{.}\) So \([8, -5, -15] \in \lspan{([1, 2, 0], [-2, 3, 5]})\text{,}\) with coefficients \(c_1 = 2\) and \(c_2 = -3\text{.}\)

    Example 8.1.8.

    Determine whether \(S = \{[1, 1], [-2, -2]\}\) is a spanning set for \(\R^2\text{.}\)
    ðŸ”—
    We have that
    \begin{align*} \lspan{(S)} \amp = \{\vect{x} \in \R^2 \;|\; \vect{x} = c_1[1, 1] + c_2[-2, -2], \quad\text{for }\, c_1, c_2 \in \R\} \\ \amp = \{\vect{x} \in \R^2 \;|\; \vect{x} = (c_1-2c_2)[1, 1], \quad\text{for }\, c_1, c_2 \in \R\} \\ \amp = \lspan{([1, 1])}\text{,} \end{align*}
    which is the line through the origin with equation \(t[1, 1]\text{,}\) for \(t\in \R\text{.}\) So any vector not on that line is not in \(\lspan{(S)}\text{.}\) We can check for example that \(\vect{e}_1 \notin \lspan{(S)}\text{;}\) if
    \begin{equation*} \vect{e}_1 = c_1[1, 1] + c_2[-2, -2] \end{equation*}
    for \(c_1, c_2 \in \R\text{,}\) which simplifies to
    \begin{equation*} [1, 0] = [c_1 - 2c_2, c_1 - 2c_2], \end{equation*}
    we can compare coefficients to get the system of linear equations
    \begin{equation*} \begin{cases} 1 \amp = c_1 - 2c_2 \\ 0 \amp = c_1 - 2c_2 \end{cases}\text{.} \end{equation*}
    This is inconsistent, so there are no solutions, so \(\vect{e}_1 \notin \lspan{(S)}\text{.}\) Hence \(S\) is not a spanning set for \(\R^2\text{.}\)
