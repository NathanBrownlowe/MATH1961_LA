    Section 4.3 Planes

    Let \(\vect{n}, \vect{x}, \vect{p} \in \R^3\text{.}\) We will investigate what the `normal formâ€™ \(\vect{n}\cdot\vect{x} = \vect{n}\cdot\vect{p}\) geometrically represents in \(\R^3\text{.}\) Consider the equation \(\vect{n}\cdot (\vect{x} - \vect{p}) = 0\text{.}\) This means that, for a fixed position vector \(\vect{p}\text{,}\) we want to find all the vectors \(\vect{x} - \vect{p}\) that are orthogonal to the `normal vectorâ€™ \(\vect{n}\text{.}\) The solutions to this describe a plane, as shown in FigureÂ 4.3.1.
    ðŸ”—

    Definition 4.3.2.

    Let \(\p\) be a plane in \(\R^3\text{.}\) A normal vector for \(\p\) is a nonzero vector \(\vect{n}\) which is orthogonal to \(\p\text{.}\)
    ðŸ”—
    ðŸ”—

    Definition 4.3.3.

    The normal form of the equation of a plane \(\p\) in \(\R^3\) is
    \begin{equation*} (\vect{x} - \vect{p}) \cdot \vect{n} =0 \end{equation*}
    or equivalently
    \begin{equation*} \vect{x} \cdot \vect{n} = \vect{p} \cdot \vect{n}, \end{equation*}
    where \(\vect{n}\) is a normal vector for \(\p\text{,}\) and \(\vect{p}\) is the position vector of a point on \(\p\text{.}\)
    ðŸ”—
    ðŸ”—
    If we know \(\vect{n} = [a, b, c]\) and \(P = (p_1, p_2, p_3)\text{,}\) so \(\vect{p} =[p_1, p_2, p_3]\text{,}\) and if we write \(X = (x, y, z)\) so \(\vect{x} = [x, y, z]\text{,}\) then evaluating the dot products in the general form \(\vect{n}\cdot \vect{x} = \vect{n}\cdot \vect{p}\) gives the equation
    \begin{equation*} ax + by + cz = d \end{equation*}
    where \(d = \vect{n}\cdot\vect{p} = ap_1 + bp_2 + cp_3\text{.}\)
    ðŸ”—

    Definition 4.3.4.

    The general form of the equation of a plane \(\p\) in \(\R^3\) is
    \begin{equation*} ax+by+cz = d \end{equation*}
    where \(\vect{n} = [a, b, c]\) is a normal vector for \(\p\text{,}\) and \(d = \vect{n}\cdot \vect{p}\text{,}\) where \(\vect{p}\) is the position vector of a point \(P\) on \(\p\text{.}\)
    ðŸ”—
    ðŸ”—

    Example 4.3.5.

    Find a normal form for the plane with general equation
    \begin{equation*} 2x - y + z = 5\text{.} \end{equation*}
    A normal vector is \(\vect{n} = [2, -1, 1]\text{.}\) A point on the plane is \(P= (0, 0, 5)\text{,}\) so \(\vect{p} = [0, 0, 5]\text{.}\) So the normal form is
    \begin{equation*} [2, -1, 1] \cdot ([x, y, z] - [0, 0, 5]) =0. \end{equation*}
    ðŸ”—
    ðŸ”—

    Example 4.3.6.

    Find a normal form and general form for the plane that passes through the point \((1, 0, 6)\) and has normal vector \([1, 2, 3]\text{.}\)
    ðŸ”—
    We take \(\vect{n} = [1, 2, 3]\) and \(P = (1, 0, 6)\text{,}\) so \(\vect{p} = [1, 0, 6]\text{.}\) The normal form is
    \begin{equation*} [1, 2, 3] \cdot [x, y, z] = [1, 2, 3] \cdot [1, 0, 6] \end{equation*}
    and the general form is \(x+2y+3z = 19\text{.}\)
    ðŸ”—
    ðŸ”—

    Remark 4.3.7.

    In \(\R^3\text{,}\) the intersection of a plane with a line can consist of zero points, one point, or infinitely many points if the line lies on the plane.
    ðŸ”—
    ðŸ”—

    Example 4.3.8.

    Determine whether the line
    \begin{equation*} [x, y, z] = [1, 2, -2] + t[2, 1, 2], \qquad t\in \R \end{equation*}
    intersects the plane \(x+2y+3z = 19\text{,}\) and if so, find the intersection.
    ðŸ”—
    The parametric equations of the line are
    \begin{equation*} \left\{ \begin{aligned} x \amp = 1+2t \\ y \amp = 2 +t \\ z \amp = -2+ 2t \end{aligned} \right. \qquad t\in\R\text{.} \end{equation*}
    Substituting this into the equation of the plane gives
    \begin{align*} \amp (1+2t) + 2(2+t) + 3(-2+2t) = 19 \\ \implies \amp 1+4-6 +(2+2+6)t = 19 \\ \implies \amp 10t = 20\text{,} \end{align*}
    so \(t = 2\text{.}\) Substituting into the parametric equations of the line gives
    \begin{equation*} x = 1+4 = 5, \qquad y = 2+2 = 4, \qquad z = -2+4 = 2, \end{equation*}
    so there is one intersection point at \((5, 4, 2)\text{.}\)
    ðŸ”—
    ðŸ”—
    We can also describe planes using direction vectors.
    ðŸ”—

    Definition 4.3.9.

    A direction vector for a plane \(\p\) in \(\R^3\) is a nonzero vector \(\vect{d} \in \R^3\) which is parallel to \(\p\text{.}\)
    ðŸ”—
    ðŸ”—
    We have previously seen that a line is specified by a point and one direction vector. In fact, a plane \(\p\) is determined by a point \(P\) on \(\p\text{,}\) and two non-parallel direction vectors \(\vect{u}\) and \(\vect{v} \in \R^3\text{.}\) (This should make sense, because intuitively a line is one dimensional, while a plane is two-dimensional, so it should have two `directionsâ€™.) From this fact, we will deduce an equation for \(\p\text{.}\) Let \(X\) be an arbitrary point of \(\p\text{.}\) Then \(\overrightarrow{PX}\) lies on \(\p\text{,}\) so there are \(s, t\in \R\) such that
    \begin{equation*} \overrightarrow{PX} = s\vect{u} + t\vect{v}\text{.} \end{equation*}
    Let \(\vect{x} =\overrightarrow{OX}\) and \(\vect{p} = \overrightarrow{OP}\text{,}\) so \(\overrightarrow{PX} = \overrightarrow{OX} - \overrightarrow{OP} = \vect{x} - \vect{p}\text{.}\) Then \(\vect{x} - \vect{p} = s\vect{u} + t\vect{v}\text{,}\) for \(s, t\in \R\text{.}\) Thus
    \begin{equation*} \vect{x} = \vect{p} + s\vect{u} + t\vect{v}, \qquad s, t\in \R\text{.} \end{equation*}
    This is shown in FigureÂ 4.3.10.
    ðŸ”—
    In components, let \(\vect{x} = [x,y, z]\text{,}\) \(\vect{p} = [p_1, p_2, p_3]\text{,}\) \(\vect{u} = [u_1, u_2, u_3]\) and \(\vect{v} = [v_1, v_2, v_3]\text{.}\) Then we have
    \begin{equation*} [x, y, z] = [p_1, p_2, p_3] + s[u_1, u_2, u_3] + t[v_1, v_2, v_3], \qquad s, t, \in\R\text{.} \end{equation*}
    Comparing coefficients gives
    \begin{equation*} \left\{ \begin{aligned} x \amp = p_1 + su_1 + tv_1 \\ y \amp = p_2 + su_2 + tv_2 \\ z \amp = p_3 + su_3 + tv_3 \end{aligned} \right.\qquad s, t\in \R\text{.} \end{equation*}
    The next definition summarises these calculations.
    ðŸ”—

    Definition 4.3.11.

    The vector form of a plane \(\p\) is
    \begin{equation*} \vect{x} = \vect{p} + s\vect{u} + t\vect{v}, \qquad s, t\in \R \end{equation*}
    and parametric equations for \(\p\) are
    \begin{equation*} \left\{ \begin{aligned} x \amp = p_1 + su_1 + tv_1 \\ y \amp = p_2 + su_2 + tv_2 \\ z \amp = p_3 + su_3 + tv_3 \end{aligned} \right.\qquad s, t\in \R \end{equation*}
    where \(\vect{p} = [p_1, p_2, p_3]\) is the position vector of a point on \(\p\text{,}\) and \(\vect{u} = [u_1, u_2, u_3]\text{,}\) \(\vect{v} = [v_1, v_2, v_3]\) are non-parallel direction vectors for \(\p\text{.}\)
    ðŸ”—
    ðŸ”—

    Remark 4.3.12.

    The reason that we require the direction vectors \(\vect{u}\) and \(\vect{v}\) for a plane to be non-parallel is because if they are parallel, then \(\vect{u} = c\vect{v}\) for some \(c\in \R\text{,}\) so
    \begin{equation*} \vect{x} = \vect{p} + s\vect{u} + t\vect{v} = \vect{p} + (sc+t)\vect{v}, \qquad s, t\in \R, \end{equation*}
    which is the equation of a line \(\ell\) in \(\R^3\text{.}\)
    ðŸ”—
    ðŸ”—
    Given any 3 non-collinear (not all on a single line) points \(P, Q, R\) on a plane \(\p\text{,}\) we can find equations for \(\p\) in any of the four forms. We can take \(\vect{u} = \overrightarrow{PQ}\) and \(\vect{v} = \overrightarrow{PR}\) to be direction vectors to find the vector form and parametric equations for \(\p\text{.}\) The vectors \(\vect{u}\) and \(\vect{v}\) are non-parallel since \(P, Q, R\) are non-collinear. Since \(\vect{u}\times\vect{v}\) is orthogonal to \(\vect{u}\) and \(\vect{v}\text{,}\) we can take
    \begin{equation*} \vect{n} = \vect{u} \times \vect{v} = \overrightarrow{PQ}\times \overrightarrow{PR} \end{equation*}
    as a normal vector to find the normal form and general form for \(\p\text{.}\)
    ðŸ”—

    Example 4.3.13.

    Find the vector form, parametric equations, normal form and general form for the plane \(\p\) containing the points \(P = (2,1, 0)\text{,}\) \(Q = (-1, 1, 2)\) and \(R = (4, 0, 2)\text{.}\)
    ðŸ”—
    We can take
    \begin{align*} \vect{p} \amp = \overrightarrow{OP} = [2, 1, 0] \\ \vect{u} \amp = \overrightarrow{PQ} = [-1, 1, 2] - [2, 1,0] = [-3, 0, 2] \\ \vect{v} \amp = \overrightarrow{PR} = [4, 0, 2] - [2, 1, 0] = [2, -1, 2]\text{.} \end{align*}
    The vector form is
    \begin{equation*} [x, y, z] = [2, 1, 0] + s[-3, 0, 2] + t[2, -1, 2], \qquad s, t\in \R \end{equation*}
    and the parametric equations are
    \begin{equation*} \left\{ \begin{aligned} x \amp = 2 - 3s + 2t \\ y \amp = 1-t \\ z \amp = 2s+2t \end{aligned} \right.\qquad s, t\in \R\text{.} \end{equation*}
    We can take
    \begin{align*} \vect{n} = \vect{u} \times\vect{v} \amp = \overrightarrow{PQ}\times \overrightarrow{PR} \\ \amp = [-3, 0, 2] \times [2, -1, 2] \\ \amp = [2, 10, 3]\text{.} \end{align*}
    Therefore the normal form is
    \begin{equation*} [2, 10, 3] \cdot [x, y, z] = [2, 10, 3]\cdot [2, 1, 0] \end{equation*}
    and the general form is \(2x+10y+3z = 14\text{.}\)
