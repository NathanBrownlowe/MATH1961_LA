    Section 7.1 Properties of inverses

    Let \(a, b, x \in \R\) and consider the equation of real numbers
    \begin{equation*} ax = b\text{.} \end{equation*}
    If \(a\ne 0\text{,}\) then \(a\) has the inverse \(a\inv = 1/a\text{,}\) and there is a unique solution
    \begin{equation*} x = a\inv b = \frac{1}{a}b = \frac{b}{a}\text{.} \end{equation*}
    This works because
    \begin{align*} ax = b \amp \implies a\inv a x = a\inv b \\ \amp \implies 1x = x = a\inv b\text{.} \end{align*}
    ðŸ”—
    We see that the key property of \(a\inv\) is that \(a\inv a = 1\text{,}\) and similarly itâ€™s useful that \(a\inv a = 1\text{.}\) (For real numbers this is essentially the same thing, because multiplication is commutative, but remember that matrix multiplication is not commutative.) We will define matrix inverses to have these properties, so that we can likewise solve the matrix equation \(AX = B\) to get \(X = A\inv B\text{.}\)
    ðŸ”—

    Definition 7.1.1.

    An inverse of an \(n\times n\) matrix \(A\) is an \(n\times n\) matrix \(B\) such that
    \begin{equation*} AB = BA = I_n\text{.} \end{equation*}
    If such a matrix \(B\) exists, then we say that \(A\) is invertible.
    ðŸ”—
    ðŸ”—
    Note that we require \(A\) and \(B\) to both be \(n\times n\) so that \(AB\) and \(BA\) are both defined.
    ðŸ”—

    Example 7.1.2.

    Let
    \begin{equation*} A = \begin{bmatrix} 2 \amp 5 \\ 1\amp 3 \end{bmatrix} , \qquad B = \begin{bmatrix} 3 \amp -5 \\ -1 \amp 2 \end{bmatrix}\text{.} \end{equation*}
    Then
    \begin{align*} AB \amp = \begin{bmatrix} 2 \amp 5 \\ 1\amp 3 \end{bmatrix} \begin{bmatrix} 3 \amp -5 \\ -1 \amp 2 \end{bmatrix} = \begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix} = I_2, \\ BA \amp = \begin{bmatrix} 3 \amp -5 \\ -1 \amp 2 \end{bmatrix} \begin{bmatrix} 2 \amp 5 \\ 1\amp 3 \end{bmatrix} = \begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix} = I_2\text{.} \end{align*}
    So by definition, \(B\) is an inverse of \(A\) (and \(A\) is an inverse of \(B\)).
    ðŸ”—
    ðŸ”—
    The zero matrix is not invertible since for any matrix \(B\text{,}\) we have \(0B = B0 = 0 \ne I_n\text{.}\) However, unlike real numbers, there are nonzero matrices which are not invertible.
    ðŸ”—

    Example 7.1.3.

    Let
    \begin{equation*} A = \begin{bmatrix} 1 \amp -1 \\ -2 \amp 2 \end{bmatrix} , \qquad B = \begin{bmatrix} a \amp b \\ c\amp d \end{bmatrix}\text{.} \end{equation*}
    Calculate \(AB\) and show that \(A\) is not invertible.
    ðŸ”—
    We have
    \begin{align*} AB \amp = \begin{bmatrix} 1 \amp -1 \\ -2 \amp 2 \end{bmatrix} \begin{bmatrix} a \amp b \\ c\amp d \end{bmatrix} \\ \amp = \begin{bmatrix} a-c \amp b-d \\ -2a+2c\amp -2(b-d) \end{bmatrix}\text{.} \end{align*}
    If the \((1, 2)\)-entry is zero, then the \((2, 2)\)-entry is also zero, so is not equal to 1. This shows that
    \begin{equation*} AB \ne I_2 = \begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix} , \end{equation*}
    regardless of the value of \(a, b, c, d\in \R\text{.}\) Therefore \(A\) has no inverse.
    ðŸ”—
    ðŸ”—

    Theorem 7.1.4.

    If an \(n\times n\) matrix \(A\) is invertible, then it has a unique inverse.
    ðŸ”—
    ðŸ”—

    Proof.

    Suppose \(B\) and \(C\) are any two inverses of \(A\text{.}\) Then
    \begin{equation*} B = BI = B(AC) \end{equation*}
    since \(C\) is an inverse of \(A\text{.}\) But then
    \begin{equation*} B(AC) = (BA)C = IC = C \end{equation*}
    using associativity and that \(B\) is an inverse of \(A\text{.}\) Thus \(B=C\text{.}\)
    ðŸ”—
    ðŸ”—
    Since if \(A\) is invertible it only has one inverse, we will now refer to it as the inverse of \(A\text{,}\) and denote it by \(A\inv\text{.}\)
    ðŸ”—

    Remark 7.1.5. .

    Do not write \(1/A\) for the inverse \(A\inv\text{.}\) This is a number divided by a matrix, so it makes no sense. Therefore, if we have a matrix equation \(AX = B\) where \(A\) is invertible, and solve it to get \(X = A\inv B\text{,}\) you should think of this as multiplication by \(A\inv\) rather than division by \(A\text{.}\) Even though for real numbers this is the same thing, to avoid confusion it is best to never think about division when working with matrices.
    ðŸ”—
    ðŸ”—

    Theorem 7.1.6.

    Suppose \(A\) and \(B\) are invertible \(n\times n\) matrices, and \(c\in \R\) is a nonzero scalar. Then
    1. \(A\inv\) is invertible, and \((A\inv)\inv = A\)
      ðŸ”—
      ðŸ”—
    2. \(cA\) is invertible, and \((cA)\inv = \frac{1}{c}A\inv\)
      ðŸ”—
      ðŸ”—
    3. \(AB\) is invertible, and \((AB)\inv = B\inv A\inv\)
      ðŸ”—
      ðŸ”—
    4. \(A^T\) is invertible, and \((A^T)\inv = (A\inv)^T\)
      ðŸ”—
      ðŸ”—
    5. \(A^k\) is invertible for any \(k \in \N\text{,}\) and \((A^k)\inv = (A\inv)^k\text{.}\)
      ðŸ”—
      ðŸ”—
    ðŸ”—
    ðŸ”—
    We will prove the first 4 properties. The proof of Property 5 is an exercise.
    ðŸ”—

    Proof.

    The equation \(AA\inv = A\inv A = I_n\) which comes from \(A\) being invertible also says that \(A\inv\) is invertible and \((A\inv)\inv = A\) (think about this) which proves 1. Property 2 follows from the calculation
    \begin{align*} cA\biggl(\frac{1}{c}A\inv\biggr) \amp = c\frac{1}{c}AA\inv = 1I = I \\ \biggl(\frac{1}{c}A\inv\biggr)cA \amp = \frac{1}{c}c A\inv A = 1I = I \end{align*}
    and Property 3 follows from the calculation
    \begin{align*} AB(B\inv A\inv) \amp = A(B B\inv)A\inv = AA\inv = I \\ (B\inv A\inv)AB \amp = B\inv (A\inv A)B = B\inv B = I\text{.} \end{align*}
    For Property 4, observe that
    \begin{align*} A^T(A\inv)^T \amp = (A\inv A)^T = I^T = I \\ (A\inv)^T A^T \amp = (AA\inv)^T = I^T = I. \end{align*}
    ðŸ”—
    ðŸ”—
    We will use the notation \(A^{-k} = (A\inv)^k\text{.}\)

    Example 7.1.7.

    Solve \(A\inv (BX)\inv = (A\inv B^3)^2\) for \(X\) in terms of \(A\) and \(B\text{.}\)
    ðŸ”—
    We calculate, using properties of the matrix inverse, that
    \begin{align*} A\inv (BX)\inv \amp = (A\inv B^3)^2 \\ \amp \implies AA\inv(BX)\inv = A(A\inv B^3)^2 \\ \amp \implies (BX)\inv = A(A\inv B^3)^2 \\ \amp \implies ((BX)\inv)\inv = (A(A\inv B^3)^2)\inv \\ \amp \implies BX = (A\inv B^3)^{-2} A\inv \\ \amp \implies B\inv BX = B\inv (A\inv B^3)^{-2} A\inv\\ \amp \implies X = B\inv(B^{-3}A)^2 A\inv = B\inv B^{-3}AB^{-3}AA\inv \\ \amp \implies X = B^{-4}AB^{-3}. \end{align*}
