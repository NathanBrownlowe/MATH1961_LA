    Section 4.1 Lines in \(\mathbb{R}^2\)

    Definition 4.1.1. (Normal vector).

    Let \(\ell\) be a line in \(\R^2\text{.}\) A normal vector for \(\ell\) is a nonzero vector \(\vect{n}\) which is orthogonal to \(\ell\text{.}\)
    ðŸ”—
    ðŸ”—
    A normal vector \(\vect{n}\) and one point \(P\) on \(\ell\) is enough to completely determine the line \(\ell\text{,}\) which we will show by writing the equation for \(\ell\) in terms of these two objects. Notice that unlike for vectors, the position of \(\ell\) in space matters, which is why we need a point \(P\text{.}\)
    ðŸ”—
    Fix \(P \in \ell\text{,}\) and let \(X = (x, y)\) be any point. Let \(\vect{x} = \overrightarrow{OX} = [x, y]\) and \(\vect{p} = \overrightarrow{OP}\text{,}\) as shown in FigureÂ 4.1.2.
    ðŸ”—
    We have
    \begin{align*} X \in \ell \amp \iff \overrightarrow{PX} \text{ is parallel to } \ell \\ \amp \iff \overrightarrow{PX} \text{ is orthogonal to } \vect{n} \\ \amp \iff \overrightarrow{PX} \cdot \vect{n} = 0 \end{align*}
    (The symbol \(\iff\) stands for `if and only ifâ€™.) We also have \(\overrightarrow{PX} = \overrightarrow{OX} - \overrightarrow{OP}\text{,}\) so
    \begin{align*} X \in \ell \amp \iff (\vect{x} - \vect{p}) \cdot \vect{n} = 0 \\ \amp \iff \vect{x} \cdot \vect{n} = \vect{p}\cdot \vect{n}\text{.} \end{align*}
    This means that \(\vect{x} \cdot \vect{n} = \vect{p}\cdot \vect{n}\) is an equation for \(\ell\text{.}\) The next definition summarises this calculation and gives the equation a name.
    ðŸ”—

    Definition 4.1.3.

    The normal form of the equation of a line \(\ell\) in \(\R^2\) is
    \begin{equation*} (\vect{x} - \vect{p}) \cdot \vect{n} = 0 \end{equation*}
    or equivalently
    \begin{equation*} \vect{x} \cdot \vect{n} = \vect{p}\cdot \vect{n} \end{equation*}
    where \(\vect{n}\) is a normal vector for \(\ell\text{,}\) and \(\vect{p}\) is the position vector of a point on \(\ell\text{.}\)
    ðŸ”—
    ðŸ”—
    You may be more familiar from high school with equations for lines looking like \(y = mx +b\) or \(ax + by = c\text{.}\) We can rearrange the normal form to look like this. Suppose \(\vect{n} = [a, b]\) and \(P = (p_1, p_2)\text{,}\) so \(\vect{p} =[p_1, p_2]\text{.}\) Then \(\vect{n}\cdot \vect{x} = \vect{n}\cdot \vect{p}\) becomes
    \begin{equation*} [a, b] \cdot [x, y] = [a, b]\cdot [p_1, p_2], \end{equation*}
    which is \(ax + by = ap_1 +bp_2\text{,}\) or \(ax + by = c\) where \(c = \vect{n}\cdot \vect{p} = ap_1 + bp_2\text{.}\)
    ðŸ”—

    Definition 4.1.4.

    The general form of the equation of a line \(\ell\) in \(\R^2\) is
    \begin{equation*} ax +by = c \end{equation*}
    where \(\vect{n} = [a, b]\) is a normal vector for \(\ell\) and \(c = \vect{n}\cdot \vect{p}\text{,}\) where \(\vect{p}\) is the position vector of a point on \(\ell\text{.}\)
    ðŸ”—
    ðŸ”—

    Example 4.1.5.

    The line with general equation \(3x- y = 9\) has a normal vector \([3, -1]\text{,}\) by looking at its coefficients. Any nonzero scalar multiple of \([3, -1]\) would also be a normal vector to the line.
    ðŸ”—
    ðŸ”—

    Example 4.1.6.

    Find the normal and general forms of the equation of a line through \((2, -1)\) and perpendicular to the vector \([3, 4]\text{.}\) We take \(\vect{n} = [3, 4]\) and \(P = (2, -1)\text{,}\) so \(\vect{p} =[2, -1]\text{.}\) We have
    \begin{equation*} \vect{n}\cdot \vect{p} = [3, 4]\cdot [2, -1] = 6 - 4 = 2, \end{equation*}
    so the normal form is \([3, 4] \cdot [x, y] = [3, 4] \cdot [2, -1]\text{,}\) which simplifies to
    \begin{equation*} [3, 4]\cdot [x, y] = 2 \end{equation*}
    and the general form is \(3x + 4y = 2\text{.}\)
    ðŸ”—
    ðŸ”—
    Notice that if we had picked a different normal vector and a different point on the same line, the normal and general forms would look different, although they would still describe the same line.
    ðŸ”—
    We will now present a different approach to finding an equation for a line in \(\R^2\text{.}\) It still involves a point and a direction. The difference is that now we will use a vector parallel to the line, instead of a vector normal to the line.
    ðŸ”—

    Definition 4.1.7.

    A direction vector for a line \(\ell\) in \(\R^2\) is a nonzero vector \(\vect{d} \in\R^2\) which is parallel to \(\ell\text{.}\)
    ðŸ”—
    ðŸ”—
    Given a direction vector \(\vect{d}\) for a line \(\ell\) and a point \(P\in \ell\) with position vector \(\vect{p}\text{,}\) we can write down an equation for \(\ell\text{.}\) Let \(X\) be any point with position vector \(\vect{x}\text{.}\) Then
    \begin{align*} X \in \ell \amp \iff \overrightarrow{PX} \text{ is parallel to } \vect{d} \\ \amp \iff \overrightarrow{PX} = t\vect{d}\text{,} \end{align*}
    for some \(t\in \R\text{.}\) We have \(\overrightarrow{PX} = \vect{x} - \vect{p}\text{,}\) so
    \begin{equation*} X \in \ell \iff \vect{x} - \vect{p} = t\vect{d} \iff \vect{x} = \vect{p} + t\vect{d}\text{.} \end{equation*}
    This is summarised in the next definition.
    ðŸ”—

    Definition 4.1.8.

    The vector form of the equation of a line \(\ell\) in \(\R^2\) which passes through a point \(P\) with position vector \(\vect{p}\text{,}\) and is parallel to a vector \(\vect{d}\text{,}\) is
    \begin{equation*} \vect{x} = \vect{p} + t\vect{d}, \end{equation*}
    where \(t \in \R\) is called a parameter.
    ðŸ”—
    ðŸ”—
    The fourth and final form of the equation of a line in \(\R^2\) comes from expanding the vector form into its components. Suppose \(\vect{d} = [d_1, d_2]\text{,}\) \(\vect{p} = [p_1, p_2]\text{,}\) and \(\vect{x} = [x, y]\text{.}\) Then we have
    \begin{equation*} [x, y] = [p_1, p_2] + t[d_1, d_2], \qquad t\in \R\text{.} \end{equation*}
    Comparing components gives
    \begin{equation*} \left\{ \begin{aligned} x \amp = p_1+ d_1t \\ y \amp = p_2 +d_2t \\ \end{aligned} \right. \qquad t\in\R\text{.} \end{equation*}
    ðŸ”—

    Definition 4.1.9.

    The parametric equations of a line in \(\R^2\) which passes through a point \(P\) with position vector \(\vect{p} = [p_1, p_2]\text{,}\) and is parallel to the vector \(\vect{d} = [d_1, d_2]\text{,}\) are
    \begin{equation*} \left\{ \begin{aligned} x \amp = p_1+ d_1t \\ y \amp = p_2 +d_2t \\ \end{aligned} \right. \qquad t\in\R\text{.} \end{equation*}
    ðŸ”—
    ðŸ”—
    Summary: A line \(\ell\) in \(\R^2\) is described by a point and a direction. If we use a normal vector to \(\ell\text{,}\) then the equation is called a normal form when written using vectors, and general form when written in components. If we use a direction vector instead, the equation is called a vector form when written using vectors, and when written in components we call them parametric equations.
    ðŸ”—
    To finish this section, we give a few examples and final comments. To find a direction vector \(\vect{d}\) for a line \(\ell\text{,}\) find two points \(A\) and \(B\) on \(\ell\text{.}\) Then \(\vect{d} = \overrightarrow{AB}\) is a direction vector for \(\ell\text{.}\) This is illustrated in FigureÂ 4.1.10.
    ðŸ”—

    Example 4.1.11.

    Find the vector form and parametric equations for the line \(\ell\) through the points \(A= (1, 2)\) and \(B = (5, 7)\text{.}\) We can take
    \begin{align*} \vect{p} \amp = \overrightarrow{OA} = [1, 2] \\ \vect{d} \amp = \overrightarrow{AB} = [5, 7] - [1, 2] = [4,5]\text{.} \end{align*}
    So a vector form is
    \begin{equation*} [x, y] = [1, 2] + t[4, 5], \qquad t\in \R \end{equation*}
    and parametric equations are
    \begin{equation*} \left\{ \begin{aligned} x \amp = 1 + 4t \\ y \amp = 2 +5t \\ \end{aligned} \right. \qquad t\in\R.\ \end{equation*}
    ðŸ”—
    ðŸ”—
    Like with the normal and general form, the expression of these equations is not unique.
    ðŸ”—
    Since a normal vector \(\vect{n}\) for a line \(\ell\) in \(\R^2\) is orthogonal, while a direction vector \(\vect{d}\) is parallel, this means
    \begin{equation*} \vect{n}\cdot \vect{d} = 0\text{.} \end{equation*}
    This observation can help us find \(\vect{d}\) if we know \(\vect{n}\text{,}\) or to find \(\vect{n}\) if we know \(\vect{d}\text{.}\) This is shown in the next two examples.
    ðŸ”—

    Example 4.1.12.

    Find a direction vector \(\vect{d}\) for the line with normal form \(x + 2y = 5\text{.}\)
    ðŸ”—
    A normal vector is \(\vect{n} = [1, 2]\text{.}\) Let \(\vect{d} = [d_1, d_2]\text{.}\) We must have
    \begin{equation*} \vect{n}\cdot \vect{d} = d_1 + 2d_2 = 0, \end{equation*}
    so \(d_1 = -2d_2\text{.}\) Choosing \(d_2 = 1\text{,}\) a direction vector is \(\vect{d} = [-2, 1]\text{.}\)
    ðŸ”—
    ðŸ”—

    Example 4.1.13.

    Find a normal vector for the line with parametric equations
    \begin{equation*} \left\{ \begin{aligned} x \amp = -1 + 2t \\ y \amp = 4 +3t \\ \end{aligned} \right. \qquad t\in\R\text{.} \end{equation*}
    A direction vector is \(\vect{d} = [2, 3]\text{,}\) so \(\vect{n} = [-3, 2]\) is a normal vector, since \(\vect{d}\cdot \vect{n} = 2(-3) + 2\cdot 3 = 0.\)
