    Section 15.1 Probability vectors and stochastic matrices

    You may know that probabilities are numbers between 0 and 1, and they add up to 1 when we consider all outcomes. This suggests that vectors with entries like this should be related to probabilities.
    ðŸ”—

    Definition 15.1.1.

    A probability vector is a vector \(\vect{x} = [x_1, x_2, \dots, x_n] \in \R^n\) such that \(0 \le x_i \le 1\) for all \(i=1, 2, \dots, n\) and
    \begin{equation*} x_1 + x_2 + \dots + x_n=1\text{.} \end{equation*}
    ðŸ”—
    ðŸ”—
    For example, the vectors
    \begin{equation*} \colvec{0.6}{0.4}, \qquad \begin{bmatrix} 1/3 \\ 1/3 \\ 1/3 \end{bmatrix} , \qquad \colvec{1}{0}, \qquad \colvec{1/3}{2/3}, \qquad \begin{bmatrix} 0 \\ 0.2\\ 0.8 \end{bmatrix} \end{equation*}
    are all probability vectors, while none of the following vectors are probability vectors:
    \begin{equation*} \colvec{-0.5}{1.5}, \qquad \colvec{0.8}{0},\qquad \colvec{1}{1}, \qquad \colvec{0}{0}\text{.} \end{equation*}
    The first is not a probability vector because it has a negative entry, and the rest are not because they do not add up to 1.
    ðŸ”—

    Definition 15.1.2.

    • A stochastic matrix is a \(n\times n\) matrix in which every column is a probability vector.
      ðŸ”—
      ðŸ”—
    • An \(n\times n\) matrix \(A = [a_{ij}]\) is positive if \(a_{ij} \gt 0\) for all \(i, j = 1, 2, \dots, n\text{.}\)
      ðŸ”—
      ðŸ”—
    • A stochastic matrix \(P\) is regular if \(P^m\) is positive for some integer \(m \ge 1\text{.}\)
      ðŸ”—
      ðŸ”—
    ðŸ”—

    Example 15.1.3.

    Determine whether the following matrices are stochastic, and if so, whether they are regular:
    \begin{equation*} A = \begin{bmatrix} 0 \amp 0.8 \\ 1 \amp 0.2 \end{bmatrix} , \qquad B = \begin{bmatrix} 0 \amp 1 \\ 1 \amp 0 \end{bmatrix}\text{.} \end{equation*}
    ðŸ”—
    First, both matrices are stochastic because all their columns have entries between \(0\) and \(1\) and sum to \(1\text{.}\) Next, \(A\) is not positive, but
    \begin{align*} A^2 = \begin{bmatrix} 0 \amp 0.8 \\ 1 \amp 0.2 \end{bmatrix} \begin{bmatrix} 0 \amp 0.8 \\ 1 \amp 0.2 \end{bmatrix} = \begin{bmatrix} 0.8 \amp 0.16 \\ 0.2 \amp 0.84 \end{bmatrix}\text{,} \end{align*}
    so \(A^2\) is positive. This shows that \(A\) is regular.
    ðŸ”—
    The matrix \(B\) is not positive. We have
    \begin{equation*} B^2 = \begin{bmatrix} 0 \amp 1 \\ 1 \amp 0 \end{bmatrix} \begin{bmatrix} 0 \amp 1 \\ 1 \amp 0 \end{bmatrix} = \begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix} = I, \end{equation*}
    which is also not positive. Then \(B^3 = B^2 B = IB = B\text{,}\) and \(B^4 = B^3 B = BB = I\text{.}\) In general,
    \begin{equation*} B^m = \begin{cases} B \amp \text{if } m \text{ is odd} \\ I \amp \text{if } m \text{ is even} \\ \end{cases}\text{.} \end{equation*}
    Therefore \(B^m\) is not positive for any power \(m\ge 1\text{,}\) and hence \(B\) is not regular.
    ðŸ”—
    ðŸ”—

    Example 15.1.4.

    The matrix
    \begin{equation*} P = \begin{bmatrix} 0.5 \amp 1 \amp 0 \\ 0.25 \amp 0 \amp 1 \\ 0.25 \amp 0 \amp 0 \end{bmatrix} \end{equation*}
    is regular since
    \begin{equation*} P^3 = \begin{bmatrix} 0.625 \amp 0.5 \amp 0.5 \\ 0.125 \amp 0.25 \amp 0.125 \\ 0.25 \amp 0.25 \amp 0.375 \end{bmatrix} \end{equation*}
    is positive.

    Theorem 15.1.5.

    Let \(P\) be a square matrix, and \(m\) a positive integer.
    1. If \(P\) is stochastic, then \(P^m\) is stochastic.
      ðŸ”—
      ðŸ”—
    2. If \(P\) is positive, then \(P^m\) is positive.
      ðŸ”—
      ðŸ”—
    ðŸ”—

    The second property is because if \(A = [a_{ij}]\) and \(B = [b_{ij}]\) are any \(n\times n\) matrices, then \(AB\) is positive. This is because \(a_{ik} \gt 0\) and \(b_{kj} \gt 0\) for every \(i, j, k = 1, \dots, n\text{,}\) so the \((i, j)\)-entry of \(AB\)
    \begin{equation*} \sum_{k=1}^n a_{ik}b_{kj} \end{equation*}
    is positive.
