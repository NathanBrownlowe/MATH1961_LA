Example 5.1.7.

Solve the system
\begin{equation*} \begin{cases} 2x+y = 2 \\ 4x + 2y = 4 \end{cases}\text{.} \end{equation*}

If we try to subtract 2 times the first equation from the second, we get \(0 = 0\text{,}\) which is true but not helpful. Instead, to solve this, let \(y = t\) where \(t\in \R\) is a parameter. Then, \(x = \frac{1}{2}(2-t)\text{,}\) \(y=t\) is a solution for any \(t\in \R\text{.}\)
