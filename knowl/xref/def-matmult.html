Definition 6.3.1.

Let \(A = [a_{ik}]_{m\times n}\) be an \(m \times n\) matrix and \(B = [b_{kj}]\) be an \(n\times p\) matrix. Then we define the matrix product
\begin{equation*} AB = [c_{ij}]_{m\times p} \end{equation*}
of size \(m\times p\) where the entries \(c_{ij}\) are defined by
\begin{equation*} c_{ij} = \sum_{k=1}^n a_{ik}b_{kj}\text{.} \end{equation*}
