Theorem 8.2.4.

A set of two vectors in \(\R^n\) is linearly independent if and only if neither vector is a scalar multiple of the other.
ðŸ”—
Equivalently, \(\{\vect{v}_1, \vect{v}_2\} \subset \R^2\) is linearly dependent if and only if at least one of the vectors is a scalar multiple of the other.
ðŸ”—
ðŸ”—

Proof.

Suppose that one of the vectors is a scalar multiple of the other; by relabelling the vectors if necessary, we can assume that
\begin{equation*} \vect{v}_2 = c\vect{v}_1 \end{equation*}
for \(c \in \R\text{.}\) Then \(c\vect{v}_1 - \vect{v}_2 = \vect{0}\text{.}\) Since the coefficient of \(\vect{v}_2\) is nonzero, the set \(\{\vect{v}_1, \vect{v}_2\}\) is linearly dependent.
ðŸ”—
Conversely, suppose that \(\{\vect{v}_1, \vect{v}_2\}\) is linearly dependent. Then there exists \(c_1, c_2 \in \R\text{,}\) not both zero, such that
\begin{equation*} c_1\vect{v}_1 + c_2\vect{v}_2 = \vect{0}\text{.} \end{equation*}
We can assume \(c_1\ne 0\text{.}\) Dividing by \(c_1\) gives \(\vect{v}_1 + \frac{c_2}{c_1}\vect{v}_2 = \vect{0}\text{,}\) so
\begin{equation*} \vect{v}_1 = -\frac{c_2}{c_1}\vect{v}_2, \end{equation*}
so \(\vect{v}_1\) is a scalar multiple of \(\vect{v}_2\text{.}\)
