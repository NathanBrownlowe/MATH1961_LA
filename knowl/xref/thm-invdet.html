Theorem 8.2.8.

An \(n\times n\) matrix \(A\) is invertible if and only if \(\det(A) \ne 0\text{.}\)

Proof.

Consider the augmented matrix \(\begin{amatrix}{c} A \amp I \end{amatrix}\) and its RREF \(\begin{amatrix}{c} R \amp B \end{amatrix} \text{.}\) If \(A\) is invertible, then there is a sequence of EROs that reduces \(A\) to \(I\text{,}\) so by applying CorollaryÂ 8.2.4, there is a constant \(k\in\R\setminus\{0\}\) such that
\begin{equation*} 1 = \det(I) = k\det(A), \end{equation*}
so \(\det(A) \ne 0\text{.}\) (Alternatively, we already showed this in TheoremÂ 8.2.6.)

On the other hand, if \(A\) is not invertible, then \(R\) must have a row of zeroes, so \(\det(R) = 0\text{.}\) Again applying CorollaryÂ 8.2.4, there is a nonzero constant \(k\) such that
\begin{equation*} 0 = \det(R)= k\det(A)\text{.} \end{equation*}
This shows that \(\det(A) = 0.\)
