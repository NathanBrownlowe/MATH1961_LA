Example 5.1.10.

Solve the following system of linear equations with an augmented matrix:
\begin{equation*} \left\{ \begin{aligned} x - y - z \amp = 2 \\ 3x - 3y + 2z \amp = 16 \\ 2x - y + z \amp = 9 \end{aligned} \right.\text{.} \end{equation*}
The corresponding augmented matrix collects all the coefficients
\begin{equation*} \begin{amatrix}{ccc} 1 \amp -1 \amp -1 \amp 2\\ 3 \amp -3 \amp 2 \amp 16\\ 2 \amp -1 \amp 1 \amp 9 \end{amatrix} \end{equation*}
To eliminate, from row two, subtract \(3 \times \text{row 1}\text{,}\) and from row three, subtract \(2\times \text{row 1}\text{.}\) This gives
\begin{equation*} \begin{amatrix}{ccc} 1 \amp -1 \amp -1 \amp 2\\ 0 \amp 0 \amp 5 \amp 10\\ 0 \amp 1 \amp 3 \amp 5 \end{amatrix} , \end{equation*}
and swapping the second and third row gives
\begin{equation*} \begin{amatrix}{ccc} 1 \amp -1 \amp -1 \amp 2\\ 0 \amp 1 \amp 3 \amp 5\\ 0 \amp 0 \amp 5 \amp 10 \end{amatrix}\text{.} \end{equation*}
We can now solve the system by `back-substitutingâ€™, which means solving the augmented matrix from the bottom upwards. The last row of the matrix says that \(5z = 10\text{,}\) so \(z = 2\text{.}\) The second row says that \(y + 3z = 5\text{,}\) so \(y = 5 - 3z = -1\text{,}\) and the first row says that \(x = 2 + y + z = 2 - 1 + 2 = 3\text{.}\)
