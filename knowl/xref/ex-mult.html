Example 12.3.2.

Find the multiplicities of the eigenvalues of
\begin{equation*} A = \begin{bmatrix} 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1\\ 2 \amp -5 \amp 4 \end{bmatrix}\text{.} \end{equation*}
ðŸ”—
We first calculate the characteristic polynomial (by expanding across the first row)
\begin{align*} \begin{vmatrix} -\lambda \amp 1 \amp 0 \\ 0 \amp -\lambda \amp 1\\ 2 \amp -5 \amp 4-\lambda \end{vmatrix} \amp = (-\lambda) \begin{vmatrix} -\lambda \amp 1 \\ -5 \amp 4-\lambda \end{vmatrix} - \begin{vmatrix} 0 \amp 1 \\ 2 \amp 4-\lambda \end{vmatrix} \\ \amp = -\lambda[-\lambda(4-\lambda) + 5] +2 \\ \amp = -\lambda^3 + 4\lambda^2 - 5\lambda +2 \\ \amp = -(\lambda - 1)^2(\lambda - 2)\text{,} \end{align*}
after factoring. So the eigenvalues are \(\lambda =1\) and \(\lambda = 2\text{.}\) The eigenvalue \(1\) has algebraic multiplicity \(2\) because of the factor \((\lambda - 1)^2\text{.}\) The eigenvalue \(2\) has algebraic multiplicity \(1\text{,}\) because it is not a repeated root.
ðŸ”—
To calculate the geometric multiplicities, we first need to calculate the eigenspaces. First we calculate for \(\lambda = 2\text{:}\)
\begin{align*} \begin{amatrix}{c} A - 2I \amp \vect{0} \end{amatrix} = \begin{amatrix}{ccc} -2 \amp 1 \amp 0 \amp 0\\ 0 \amp -2 \amp 1 \amp 0\\ 2 \amp -5 \amp 2 \amp 0 \end{amatrix} \xrightarrow{R_3\mapsto R_3 + R_1} \amp \begin{amatrix}{ccc} -2 \amp 1 \amp 0 \amp 0\\ 0 \amp -2 \amp 1 \amp 0\\ 0 \amp -4 \amp 2 \amp 0 \end{amatrix} \\ \xrightarrow{R_3 \mapsto R_3 - 2R_2} \amp \begin{amatrix}{ccc} -2 \amp 1 \amp 0 \amp 0\\ 0 \amp -2 \amp 1 \amp 0\\ 0 \amp 0 \amp 0 \amp 0 \end{amatrix}\text{.} \end{align*}
Let \(z = t\text{,}\) \(t \in \R\text{.}\) Then \(y = t/2\) and row 1 gives that \(-2x+y = 0\text{,}\) so \(x = t/4\text{.}\) So
\begin{equation*} E_2(A) = \left\{ \begin{bmatrix} \frac{t}{4} \\ \frac{t}{2}\\ t \end{bmatrix} \;\middle|\; t \in \R \right\} = \left\{ \begin{bmatrix} t \\ 2t\\ 4t \end{bmatrix} \;\middle|\; t \in \R \right\}\text{.} \end{equation*}
This has 1 parameter, so the geometric multiplicity of \(\lambda =2\) is \(1\text{.}\)

For \(\lambda = 1\text{,}\) we solve
\begin{align*} \begin{amatrix}{c} A -I \amp \vect{0} \end{amatrix} = \begin{amatrix}{ccc} -1 \amp 1 \amp 0 \amp 0\\ 0 \amp -1 \amp 1 \amp 0\\ 2 \amp -5 \amp 3 \amp 0 \end{amatrix} \xrightarrow{R_3\mapsto R_3 + 2R_1} \amp \begin{amatrix}{ccc} -1 \amp 1 \amp 0 \amp 0\\ 0 \amp -1 \amp 1 \amp 0\\ 0 \amp -3 \amp 3 \amp 0 \end{amatrix} \\ \xrightarrow{R_3 \mapsto R_3 - 3R_2} \amp \begin{amatrix}{ccc} -1 \amp 1 \amp 0 \amp 0\\ 0 \amp -1 \amp 1 \amp 0\\ 0 \amp 0 \amp 0 \amp 0 \end{amatrix}\text{.} \end{align*}
Let \(z = t, t\in \R\text{.}\) Then \(y = t\) and \(x = t\) also. So the eigenspace is \(E_1(A) = \{[t, t, t] \;|\; t\in \R\}\text{,}\) which has 1 parameter, so the geometric multiplicity of \(\lambda = 1\) is \(1\text{.}\) The answer is summarised in .
