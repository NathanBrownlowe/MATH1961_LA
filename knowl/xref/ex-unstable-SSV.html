First, the transition matrix is
\begin{equation*} P = \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1\\ 0 \amp 1 \amp 0 \end{bmatrix}\text{.} \end{equation*}
Solving \((P-I)\vect{x} = \vect{0}\text{,}\) we get
\begin{equation*} \begin{amatrix}{c} P - I \amp 0 \end{amatrix} = \begin{amatrix}{ccc} 0 \amp 0 \amp 0 \amp 0\\ 0 \amp -1 \amp 1 \amp 0\\ 0 \amp 1 \amp -1 \amp 0 \end{amatrix} \xrightarrow{R_3 \mapsto R_3 + R_2} \begin{amatrix}{ccc} 0 \amp 0 \amp 0 \amp 0\\ 0 \amp -1 \amp 1 \amp 0\\ 0 \amp 0 \amp 0 \amp 0 \end{amatrix}\text{.} \end{equation*}
Let \(x = s, z = t\text{.}\) Then the second row gives \(y=z=t\text{.}\) So the solutions are
\begin{equation*} \left\{s \begin{bmatrix} 1 \\ 0 \\ 0 \end{bmatrix} + t \begin{bmatrix} 0 \\ 1\\ 1 \end{bmatrix} \;\middle| \; s, t \in \R\right\}\text{.} \end{equation*}
So there are two different steady state vectors, \([600, 0, 0]\) (where \(s = 600\text{,}\) \(t=0\)) and \([0, 300, 300]\) (where \(s = 0\) and \(t = 300\)). There are also two different SSPVs \([1, 0, 0]\) (from \(s=1\text{,}\) \(t=0\)) and \([0, 1/2, 1/2]\) (from \(s=0, t=1/2)\text{.}\)

Notice that if the initial state vector is
\begin{equation*} \vect{x}_0 = \begin{bmatrix} 600 \\ 0 \\ 0 \end{bmatrix} , \qquad \text{then }\, \vect{x}_k = \begin{bmatrix} 600 \\ 0 \\ 0 \end{bmatrix} \, \text{for all } k\ge 1, \end{equation*}
because \(\vect{x}_0\) is a SSV. But if the initial state vector is \(\vect{x}_0 = [0, 300, 300]\text{,}\) the other SSV, then \(\vect{x}_k = [0, 300, 300]\) for all \(k\ge 1\) instead.
