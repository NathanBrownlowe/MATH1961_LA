    Section 3.1 Cross products

    The cross product is another operation on vectors. It is only defined for vectors in \(\R^3\text{,}\) and its output is another vector in \(\R^3\text{.}\)
    ðŸ”—

    Definition 3.1.1.

    The cross product of the vectors \(\vect{u}=[u_1, u_2, u_3], \vect{v} = [v_1, v_2, v_3] \in \R^3\) is the vector
    \begin{equation*} \vect{u}\times\vect{v} = [u_2v_3 - u_3v_2, u_3v_1 - u_1v_3, u_1v_2-u_2v_1]\text{.} \end{equation*}
    ðŸ”—
    ðŸ”—
    To help remember the formula, we can use the array method (FigureÂ 3.1.2). It involves 3 pairs of lines, one for each component of the resulting vector. Starting at \(u_2\text{,}\) multiply the components connected by the solid line, then subtract the product of the components connected by the dashed line. For example, the first component is
    \begin{equation*} \text{first solid line} - \text{first dashed line} = u_2v_3 - v_2u_3, \end{equation*}
    and so on for the rest.
    ðŸ”—
    Alternatively, to notice the `criss-crossâ€™ pattern it helps to write the vectors using columns:
    \begin{equation*} \begin{bmatrix} u_1 \\ u_2\\ u_3 \end{bmatrix} \times \begin{bmatrix} v_1 \\ v_2\\ v_3 \end{bmatrix} = \begin{bmatrix} u_2v_3 - u_3v_2 \\ u_3v_1 - u_1v_3\\ u_1v_2-u_2v_1 \end{bmatrix}\text{.} \end{equation*}
    ðŸ”—
    Here are three examples which also showcase properties of the cross product.
    ðŸ”—

    Example 3.1.3.

    If \(\vect{u} = [3, 1, 2]\) and \(\vect{v} = [5, 0, 4]\text{,}\) then
    \begin{equation*} \vect{u} \times \vect{v} = [1(4) - 2(0), 2(5) - 3(4), 3(0) - 1(5)] = [4, -2, -5] \end{equation*}
    and
    \begin{equation*} \vect{v}\times\vect{u} = [0(2) - 4(1), 4(3) - 5(2), 5(1) - 0(3)] = [-4, 2, 5]\text{.} \end{equation*}
    Notice that \(\vect{u} \times \vect{v} = - \vect{v} \times \vect{u}\text{.}\)
    ðŸ”—
    ðŸ”—

    Example 3.1.5.

    Find \(\vect{u}\cdot (\vect{u}\times\vect{v})\) and \(\vect{v}\cdot (\vect{u}\times\vect{v})\) for \(\vect{u} = [3, 1, 2]\) and \(\vect{v} = [5, 0, 4]\text{.}\) From the previous example, we know \(\vect{u} \times\vect{v} = [4, -2, -5]\text{.}\) So
    \begin{equation*} \vect{u}\cdot(\vect{u}\times\vect{v}) = [3, 1, 2] \cdot [4, -2, -5] = 12-2-10 = 0, \end{equation*}
    \begin{equation*} \vect{v}\cdot (\vect{u}\times\vect{v}) = [5, 0, 4] \cdot [4, -2, -5] = 20+0-20 = 0. \end{equation*}
    ðŸ”—
    ðŸ”—

    Example 3.1.6.

    If \(\vect{u} = [3, 1, 2]\text{,}\) then
    \begin{equation*} \vect{u}\times \vect{u} = [2-2, 6-6, 3-3] = [0, 0, 0] = \vect{0}. \end{equation*}
    ðŸ”—
    ðŸ”—
    The next theorem lists the main properties of the cross product.
    ðŸ”—

    Theorem 3.1.7.

    Let \(\vect{u}, \vect{v}, \vect{w} \in\R^3\) and \(c\in \R\text{.}\) Then
    1. \(\vect{u} \times \vect{v} = -\vect{v} \times \vect{u}\quad\) (anti-commutativity)
      ðŸ”—
      ðŸ”—
    2. \(\displaystyle \vect{u} \cdot (\vect{u}\times \vect{v}) = \vect{v} \cdot(\vect{u}\times \vect{v} ) = 0\)
      ðŸ”—
      ðŸ”—
    3. \(\displaystyle \vect{u} \times \vect{u} = \vect{0}\)
      ðŸ”—
      ðŸ”—
    4. \(\displaystyle \vect{u} \times \vect{0} = \vect{0} \times\vect{u} = \vect{0}\)
      ðŸ”—
      ðŸ”—
    5. \(\vect{u}\times (\vect{v} + \vect{w})=\vect{u}\times \vect{v} + \vect{u} \times \vect{w}\quad\) (left distributivity)
      ðŸ”—
      ðŸ”—
    6. \((\vect{u} + \vect{v})\times \vect{w} = \vect{u} \times\vect{w} + \vect{v} \times \vect{w}\quad\) (right distributivity)
      ðŸ”—
      ðŸ”—
    7. \((c\vect{u})\times \vect{v} = \vect{u} \times (c\vect{v}) = c(\vect{u}\times \vect{v})\text{.}\)
      ðŸ”—
      ðŸ”—
    ðŸ”—
    ðŸ”—
    Property 2 says that \(\vect{u} \times \vect{v}\) is orthogonal to \(\vect{u}\) and \(\vect{v}\text{.}\) We will prove Properties 2, 3, and 5.
    ðŸ”—

    Proof.

    Let \(\vect{u} = [u_1, u_2, u_3], \vect{v} = [v_1, v_2, v_3]\) and \(\vect{w} = [w_1, w_2, w_3]\text{.}\) Property 2 holds since
    \begin{align*} \vect{u} \cdot (\vect{u}\times \vect{v}) \amp = [u_1, u_2, u_3]\cdot [u_2v_3 - u_3v_2, u_3v_1 - u_1v_3, u_1v_2-u_2v_1] \\ \amp = u_1u_2v_3 - u_1u_3v_2 + u_2u_3v_1 - u_2u_1v_3 + u_3u_1v_2 - u_3u_2v_1 \\ \amp =0; \end{align*}
    the calculation that \(\vect{v} \cdot(\vect{u}\times \vect{v}) = 0\) is similar. We can deduce Property 3 from Property 1 by observing that
    \begin{equation*} \vect{u} \times \vect{u} = -\vect{u} \times \vect{u}, \end{equation*}
    so \(\vect{u} \times \vect{u} = 0\text{.}\) Finally, the proof of Property 5 is that
    \begin{align*} \amp\vect{u}\times (\vect{v} + \vect{w}) \\ \amp = [u_1, u_2, u_3]\times [v_1 + w_1, v_2 + w_2, v_3 + w_3] \\ \amp = [u_2(v_3 + w_3) - u_3(v_2 + w_2), u_3(v_1 + w_1)- u_1(v_3 + w_3), \\ \amp\qquad\qquad u_1(v_2 + w_2) - u_2(v_1 + w_1)] \\ \amp = [u_2v_3 - u_3v_2, u_3v_1 - u_1v_3, u_1v_2 - u_2v_1] \\ \amp\quad\qquad+ [u_2w_3 - u_3w_2, u_3w_1 - u_1w_3, u_1w_2 - u_2w_1] \\ \amp = \vect{u} \times \vect{v} + \vect{u} \times \vect{w}. \end{align*}
    ðŸ”—
    ðŸ”—
    Importantly, the cross product is not associative! This means that, for \(\vect{u}, \vect{v}, \vect{w} \in \R^3\text{,}\)
    \begin{equation*} (\vect{u}\times \vect{v})\times \vect{w} \ne \vect{u} \times (\vect{v}\times \vect{w})\text{.} \end{equation*}
    For example, if \(\vect{u} = [1, 1, 0], \vect{v} = [0, 1, 1], \vect{w} = [0, 0, 1]\text{,}\) then
    \begin{equation*} (\vect{u}\times \vect{v})\times \vect{w} = [1, -1, 1] \times [0, 0, 1] =[-1, -1, 0] \end{equation*}
    while
    \begin{equation*} \vect{u} \times (\vect{v}\times \vect{w}) = [1, 1, 0] \times [1, 0, 0] = [0, 0, -1]\text{.} \end{equation*}
    ðŸ”—

    Example 3.1.8.

    Simplify \((\vect{u} + \vect{v}) \times (\vect{u} - \vect{v})\) using the properties in the theorem.
    \begin{align*} (\vect{u} + \vect{v}) \times (\vect{u} - \vect{v}) \amp = (\vect{u} + \vect{v}) \times \vect{u} + (\vect{u} + \vect{v}) \times (-\vect{v})\quad \text{(Property 5)} \\ \amp = (\vect{u} + \vect{v}) \times \vect{u} - (\vect{u} + \vect{v})\times\vect{v} \quad \text{(Property 7)} \\ \amp = \vect{u}\times \vect{u} + \vect{v} \times \vect{u} - (\vect{u} \times \vect{v} + \vect{v}\times \vect{v}) \quad \text{(Property 6)} \\ \amp = \vect{v} \times \vect{u} - \vect{u} \times \vect{v} \quad \text{(Property 3)} \\ \amp = - 2\vect{u}\times \vect{v}. \quad \text{(Property 1)} \end{align*}
    ðŸ”—
    ðŸ”—
    Recall that \(\vect{u} \times \vect{v}\) is orthogonal to both \(\vect{u}\) and \(\vect{v}\text{.}\) If \(\vect{u}\) and \(\vect{v}\) are not parallel, then the set of linear combinations of \(\vect{u}\) and \(\vect{v}\) is a plane in \(\R^3\text{.}\) The cross product \(\vect{u} \times \vect{v}\) must be orthogonal to this plane, so there are only two choices for its direction. This is show in FigureÂ 3.1.9: the two orthogonal vectors are the upwards normal vector and the downwards normal vector, drawn in red.
    ðŸ”—
    The direction of \(\vect{u} \times \vect{v}\) is given by the right hand rule.
    ðŸ”—
    ðŸ”—
    Now we know the direction of \(\vect{u}\times \vect{v}\text{.}\) Next, we want to calculate its magnitude (length). First, it will help to have the following fact (we leave the proof as an exercise).
    ðŸ”—
    Let \(\vect{u}, \vect{v} \in \R^3\text{.}\) Then \(\|\vect{u} \times\vect{v}\|^2 = \|\vect{u}\|^2\|\vect{v}\|^2 - |\vect{u}\cdot\vect{v}|^2\text{.}\)
    ðŸ”—
    ðŸ”—
    Now we can calculate the length of a cross product.
    ðŸ”—

    Theorem 3.1.11.

    Let \(\vect{u}, \vect{v} \in \R^3\text{,}\) and \(\theta\) be the angle between \(\vect{u}\) and \(\vect{v}\) such that \(0 \le \theta \le \pi\text{.}\) Then
    \begin{equation*} \|\vect{u} \times \vect{v}\| = \|\vect{u}\|\|\vect{v}\|\sin\theta\text{.} \end{equation*}
    ðŸ”—
    ðŸ”—

    Proof.

    By the boxed fact above and the geometric formula for the dot product,
    \begin{align*} \|\vect{u} \times\vect{v}\|^2 \amp = \|\vect{u}\|^2\|\vect{v}\|^2 - |\vect{u}\cdot\vect{v}|^2 \\ \amp = \|\vect{u}\|^2\|\vect{v}\|^2 - |\|\vect{u}\|\|\vect{v}\|\cos\theta|^2 \\ \amp = \|\vect{u}\|^2\|\vect{v}\|^2(1-\cos^2\theta) \\ \amp = \|\vect{u}\|^2\|\vect{v}\|^2\sin^2\theta\text{.} \end{align*}
    Since \(0 \le \theta\le \pi\text{,}\) we have \(\sin\theta \ge 0\text{,}\) and so taking square roots gives the result.
    ðŸ”—
    ðŸ”—

    Example 3.1.12.

    Suppose \(\vect{u}, \vect{v} \in \R^3\) are such that \(\|\vect{u}\| = \sqrt{6}, \|\vect{v}\| = \sqrt{35}\text{,}\) and \(\vect{u}\cdot\vect{v} = 14\text{.}\) Find \(\|\vect{u} \times \vect{v}\|\text{.}\) We first calculate that
    \begin{equation*} \cos\theta = \frac{\vect{u}\cdot\vect{v}}{\|\vect{u}\|\|\vect{v}\|} = \frac{14}{\sqrt{6}\sqrt{35}} = \frac{14}{\sqrt{210}}\text{.} \end{equation*}
    Then \(\sin^2\theta = 1- \cos^2\theta = \frac{1}{15}\text{,}\) so \(\sin\theta = \frac{1}{\sqrt{15}}\text{.}\) (We used the positive square root since \(0\le\theta\le\pi\)). Finally,
    \begin{equation*} \|\vect{u} \times \vect{v}\| = \|\vect{u}\|\|\vect{v}\|\sin\theta = \sqrt{\frac{210}{15}} = \sqrt{14}. \end{equation*}
