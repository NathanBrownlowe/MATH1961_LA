    Section 6.2 Matrix addition and scalar multiplication

    Definition 6.2.1.

    The sum of two \(m\times n\) matrices
    \begin{equation*} A = [a_{ij}]_{m\times n}, \qquad B = [b_{ij}]_{m\times n} \end{equation*}
    is the \(m\times n\) matrix
    \begin{equation*} A + B = [a_{ij} + b_{ij}]_{m\times n}\text{.} \end{equation*}

    The scalar multiple of an \(m\times n\) matrix \(A = [a_{ij}]_{m\times n}\) by a scalar \(c \in \R\) is the \(m \times n\) matrix
    \begin{equation*} cA = [ca_{ij}]_{m\times n}\text{.} \end{equation*}
