    Section 15.3 Steady state vectors

    Suppose that we model the weather in Sydney by the three states 1: sunny, 2: cloudy, and 3: rainy, with transition probabilities given by the diagram FigureÂ 15.3.1.
    ðŸ”—
    The transition matrix is therefore
    \begin{equation*} P = \begin{bmatrix} 0 \amp 0.25 \amp 0.25 \\ 0.5 \amp 0.25 \amp 0.5 \\ 0.5 \amp 0.5 \amp 0.25 \end{bmatrix}\text{.} \end{equation*}
    ðŸ”—
    Suppose the weather is rainy (state 3) on day 0, so the initial state vector is
    \begin{equation*} \vect{x}_0 = \begin{bmatrix} 0 \\ 0\\ 1 \end{bmatrix}\text{.} \end{equation*}
    What are the weather probabilities on day 2? They are
    \begin{equation*} \vect{x}_2 = P^2\vect{x}_0 = \begin{bmatrix} 0 \amp 0.25 \amp 0.25 \\ 0.5 \amp 0.25 \amp 0.5 \\ 0.5 \amp 0.5 \amp 0.25 \end{bmatrix} ^2 \begin{bmatrix} 0 \\ 0\\ 1 \end{bmatrix} = \begin{bmatrix} 0.1875 \\ 0.375\\ 0.4375 \end{bmatrix}\text{.} \end{equation*}
    So on day 2 the probability of sunny weather is 0.1875, of cloudy weather is 0.375, and of rainy weather is 0.4375. It is possible to calculate that in 7 days time
    \begin{equation*} \vect{x}_7 = P^7\vect{x}_0 = \begin{bmatrix} 0.2000122070 \\ 0.4000244140\\ 0.3999633789 \end{bmatrix} \end{equation*}
    and that in 15 days time
    \begin{equation*} \vect{x}_{15} = P^{15}\vect{x}_0 = \begin{bmatrix} 0.2000000002 \\ 0.4000000003\\ 0.3999999994 \end{bmatrix}\text{.} \end{equation*}
    It seems like
    \begin{equation*} \vect{x}_k \to \begin{bmatrix} 0.2 \\ 0.4\\ 0.4 \end{bmatrix} \end{equation*}
    as \(k\to \infty\text{.}\) Let \(\vect{x} = \begin{bmatrix} 0.2, 0.4, 0.4 \end{bmatrix} \text{.}\) We calculate
    \begin{equation*} P\vect{x} = \begin{bmatrix} 0 \amp 0.25 \amp 0.25 \\ 0.5 \amp 0.25 \amp 0.5 \\ 0.5 \amp 0.5 \amp 0.25 \end{bmatrix} \begin{bmatrix} 0.2 \\ 0.4\\ 0.4 \end{bmatrix} = \begin{bmatrix} 0 + 0.1 + 0.1 \\ 0.1 + 0.1 + 0.2 \\ 0.1 + 0.2 + 0.1 \end{bmatrix} = \begin{bmatrix} 0.2 \\ 0.4\\ 0.4 \end{bmatrix} = \vect{x}\text{.} \end{equation*}
    Therefore \(P\vect{x} = \vect{x}\text{,}\) or in other words \(\vect{x}\) is an eigenvector of \(P\) with eigenvalue \(1\text{.}\) It is an example of a steady state probability vector, which we define next.
    ðŸ”—

    Definition 15.3.2.

    Let \(P\) be the transition matrix of a Markov chain. A steady state vector (SSV) is a vector \(\vect{x}\) such that \(P\vect{x} = \vect{x}\text{,}\) with entries which are non-negative and sum to the total number of members in the Markov chain. A steady state probability vector (SSPV) is a probability vector \(\vect{x}\) such that \(P\vect{x} = \vect{x}\text{.}\)
    ðŸ”—
    ðŸ”—

    Remark 15.3.3.

    If \(\vect{x}\) is a SSV or SSPV, then \(P\vect{x} = \vect{x}\text{.}\) Thus \(P^2 \vect{x} = P(P\vect{x)} = P\vect{x} = \vect{x}\text{,}\) and
    \begin{equation*} P^3\vect{x} = P(P^2\vect{x}) = P\vect{x} = \vect{x}, \end{equation*}
    and so on. Therefore \(P^k\vect{x} = \vect{x}\) for all integers \(k\ge 1\text{.}\)
    ðŸ”—
    ðŸ”—

    Theorem 15.3.4.

    If \(P\) is a transition matrix for a Markov chain, then \(1\) is an eigenvalue of \(P\text{.}\)
    ðŸ”—
    ðŸ”—
    This theorem is important because it means that there is a nonzero vector \(\vect{y}\) such that \(P\vect{y} = \vect{y}\text{.}\) We can scale \(\vect{y}\) to make it a SSV or SSPV.
    ðŸ”—

    Proof.

    Let \(\vect{r} = \begin{bmatrix} 1 \amp 1 \amp \dots \amp 1 \end{bmatrix}\) be the \(1\times n\) row matrix whose entries are all 1s. Then
    \begin{align*} \vect{r}P \amp = \begin{bmatrix} 1 \amp 1 \amp \dots \amp 1 \end{bmatrix} \begin{bmatrix} p_{11} \amp p_{12} \amp \cdots \amp p_{1n} \\ p_{21} \amp p_{22} \amp \cdots \amp p_{2n} \\ \vdots \amp \vdots \amp \amp \vdots \\ p_{n1} \amp p_{n2} \amp \cdots \amp p_{nn} \end{bmatrix} \\ \amp = \begin{bmatrix} p_{11} + p_{12} + \dots + p_{1n} \amp \dots \amp p_{n1} + p_{n2}+\dots +p_{nn} \end{bmatrix} \\ \amp = \begin{bmatrix} 1 \amp 1 \amp \cdots \amp 1 \end{bmatrix} \\ \amp = \vect{r}\text{.} \end{align*}
    ðŸ”—
    Here we used that \(p_{i1} + p_{i2} + \dots + p_{in}\text{,}\) the sum of the \(i\)th column of \(P\text{,}\) is \(1\) since \(P\) is stochastic. This shows that \(\vect{r}P = \vect{r}\text{.}\) Taking transposes gives that \(P^T\vect{r}^T = \vect{r}^T\text{,}\) so \(\vect{r}^T\) is an eigenvector of \(P^T\) with eigenvalue 1. Since
    \begin{equation*} \det(P^T - \lambda I) = \det((P-\lambda I)^T) = \det(P-\lambda I), \end{equation*}
    the matrices \(P\) and \(P^T\) have the same characteristic polynomial, and hence the same eigenvalues. This proves that there is an eigenvector of \(P\) with eigenvalue 1.
    ðŸ”—
    ðŸ”—
    To find a SSV or SSPV, solve the equation
    \begin{equation*} (P-I)\vect{x} = \vect{0}\text{.} \end{equation*}
    By the previous theorem, this matrix equation has a nonzero solution, and therefore infinitely many solutions. By substituting the appropriate value of the parameter, we can make the vector add up to the total number of members in the Markov chain to get a SSV, or add up to 1 to get a SSPV.
    ðŸ”—

    Example 15.3.5.

    Consider the grocery store Markov chain from ExampleÂ 15.2.1 with transition matrix and initial state vector
    \begin{equation*} P = \begin{bmatrix} 0.9 \amp 0.3 \\ 0.1 \amp 0.7 \end{bmatrix} , \qquad \vect{x}_0 = \colvec{100}{80}\text{.} \end{equation*}
    Find a steady state vector and a steady state probability vector.
    ðŸ”—
    First we solve \((P-I)\vect{x} = \vect{0}\) to get
    \begin{equation*} \begin{amatrix}{c} P-I \amp 0 \end{amatrix} = \begin{amatrix}{cc} -0.1 \amp 0.3 \amp 0\\ 0.1 \amp -0.3 \amp 0 \end{amatrix} \xrightarrow{R_2 \mapsto R_2 + R_1} = \begin{amatrix}{cc} -0.1 \amp 0.3 \amp 0\\ 0 \amp 0 \amp 0 \end{amatrix}\text{.} \end{equation*}
    Let \(y = t\text{,}\) then \(x = 3t\text{.}\) So the solutions are given by the 1-eigenspace
    \begin{equation*} E_1(P) = \left\{t\colvec{3}{1} \;\middle|\; t\in \R\right\}\text{.} \end{equation*}
    The total population is \(100 + 80 = 180\text{,}\) so a SSV is a vector in \(E_1(P)\) with nonnegative entries that add up to 180. This means that \(3t + t = 180\text{,}\) so \(t = 180/4 = 45\text{.}\) Substituting in \(t=45\) gives that a SSV is \([135, 45]\text{.}\) To find a SSPV, we solve for \(3t+t = 1\text{,}\) so \(t=1/4\text{.}\) Therefore a SSPV is \([3/4, 1/4].\)
