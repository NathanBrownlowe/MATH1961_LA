    Definition 5.2.4.

    Gaussian elimination is the process of solving a system of linear equations by applying elementary row operations to transform the corresponding augmented matrix to row echelon form, and then using back-substitution (solving the augmented matrix from the bottom row upwards).
    ðŸ”—
    ðŸ”—
    Here are steps to row reduce a matrix into REF:
    1. By swapping rows, make sure that the top row has less (or the same amount of) leading zeroes than all the other rows.
      ðŸ”—
      ðŸ”—
    2. Then use the top row to make all the elements below its leading entry zero, by subtracting multiples of this row.
      ðŸ”—
      ðŸ”—
    3. Once you have cleared out the first column, you are finished with the first row and column, so they should stay where they are. Move onto the second row and repeat the process, now using the second row to make all the entries below its leading entry zero.
      ðŸ”—
      ðŸ”—
    4. Repeat this for every row to get the matrix into REF.
      ðŸ”—
      ðŸ”—
    To perform Gaussian elimination, follow the steps to perform REF and then back-substitute. To understand and remember these steps, it is much easier to learn by example than to directly memorise them, so we will give many worked examples throughout the rest of this chapter. (Also, in some examples there may be quicker ways to get the matrix into REF. You donâ€™t need to follow the steps above exactly: as long as the matrix ends up in REF itâ€™s fine.)

    Example 5.2.5.

    Solve the system of linear equations
    \begin{equation*} \begin{cases} x +y + 3z \amp = 4 \\ 2x + 3y \amp = 1 \\ -x + z \amp = 9 \end{cases}\text{.} \end{equation*}
    ðŸ”—
    We form the augmented matrix and row-reduce:
    \begin{align*} \begin{amatrix}{ccc} 1 \amp 1 \amp 3 \amp 4\\ 2 \amp 3 \amp 0 \amp 1\\ -1 \amp 0 \amp 1 \amp 9\\ \end{amatrix} \amp\xrightarrow[R_3\mapsto R_3 + R_1]{R_2 \mapsto R_2 - 2R_1} \begin{amatrix}{ccc} 1 \amp 1 \amp 3 \amp 4\\ 0 \amp 1 \amp -6 \amp -7\\ 0 \amp 1 \amp 4 \amp 13\\ \end{amatrix} \\ \amp\xrightarrow{R_3 \mapsto R_3 - R_2} \begin{amatrix}{ccc} 1 \amp 1 \amp 3 \amp 4\\ 0 \amp 1 \amp -6 \amp -7\\ 0 \amp 0 \amp 10 \amp 20 \end{amatrix}\text{.} \end{align*}
    Back-substituting, the bottom row gives \(10z = 20\text{,}\) so \(z = 2\text{.}\) The next row up (the middle row) gives \(y - 6z = -7\text{,}\) so \(y = -7 + 6z = -7 + 12 = 5\text{.}\) The top row gives \(x + y + 3z = 4\text{,}\) so \(x = 4 - y - 3z = -7\text{.}\) Therefore the unique solution is \(x = -7\text{,}\) \(y = 5\text{,}\) \(z = 2\text{.}\)

    Example 5.2.6.

    Use Gaussian elimination to solve the following system of linear equations:
    \begin{equation*} \left\{ \begin{aligned} 3x +y + 2z \amp = 13 \\ -3x - y - z \amp = -6 \\ -3x + 5y - 4z \amp = 3 \end{aligned} \right.\text{.} \end{equation*}
    ðŸ”—
    Calculating REF:
    \begin{align*} \begin{amatrix}{ccc} 3 \amp 1 \amp 2 \amp 13\\ -3 \amp -1 \amp -1 \amp -6\\ -3 \amp 5 \amp -4 \amp 3\\ \end{amatrix} \amp\xrightarrow[R_3\mapsto R_3 + R_1]{R_2 \mapsto R_2 + R_1} \begin{amatrix}{ccc} 3 \amp 1 \amp 2 \amp 13\\ 0 \amp 0 \amp 1 \amp 7\\ 0 \amp 6 \amp -2 \amp 16 \end{amatrix} \\ \amp\xrightarrow{R_2 \leftrightarrow R_3} \begin{amatrix}{ccc} 3 \amp 1 \amp 2 \amp 13\\ 0 \amp 6 \amp -2 \amp 16\\ 0 \amp 0 \amp 1 \amp 7 \end{amatrix}\text{.} \end{align*}
    Back-substituting, \(z = 7\text{.}\) Then \(6y - 2z = 16\text{,}\) so \(y = \frac{1}{6}(16+2z) = \frac{1}{6}(16+14) = 5\text{.}\) Finally, the first row gives \(3x+y+2z=13\text{,}\) which rearranges to give \(x = -2\text{.}\)

    Remark 5.2.7.

    REF is not unique. For example, if an augmented matrix is in REF, we can multiply any of its rows by a nonzero scalar. This will give a different row echelon form of the matrix. REF matrices can have other differences too.
