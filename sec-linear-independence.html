    Section 8.2 Linear independence

    Definition 8.2.1.

    Let \(S = \{\vect{v}_1, \vect{v}_2, \dots, \vect{v}_k\}\) be a set of vectors in \(\R^n\text{.}\) The set \(S\) is linearly independent if the only solution to the equation
    \begin{equation*} c_1\vect{v}_1 + c_2\vect{v}_2 + \dots + c_k\vect{v}_k = \vect{0} \end{equation*}
    for \(c_1, \dots, c_k \in \R\) is
    \begin{equation*} c_1 = c_2 = \cdots = c_k = 0\text{.} \end{equation*}
    The set \(S\) is linearly dependent if it is not linearly independent.
    ðŸ”—
    ðŸ”—
    So a set \(S = \{\vect{v}_1, \vect{v}_2, \dots, \vect{v}_k\}\) is linearly dependent if and only if there exists scalars \(c_1, c_2, \dots, c_k \in \R\text{,}\) not all zero, such that
    \begin{equation*} c_1\vect{v}_1 + c_2\vect{v}_2 + \dots + c_k\vect{v}_k = \vect{0}\text{.} \end{equation*}
    ðŸ”—

    Example 8.2.2.

    Let \(\vect{u} = [2, 1]\text{,}\) \(\vect{v} = [1, -1]\text{,}\) and \(\vect{w} = [1, 2]\text{.}\) Find \(\vect{u} - \vect{v}\text{,}\) and explain why the set \(\{\vect{u}, \vect{v}, \vect{w}\}\) is linearly dependent.
    ðŸ”—
    We have
    \begin{equation*} \vect{u} - \vect{v} = [2, 1] - [1, -1] = [1, 2] = \vect{w}, \end{equation*}
    so \(\vect{u} - \vect{v} - \vect{w} = \vect{0}\text{.}\) This is a linear combination of \(\vect{u}, \vect{v}, \vect{w}\) equal to zero, where at least one (in fact all) of the coefficients are nonzero. Therefore, the set \(\{\vect{u}, \vect{v}, \vect{w}\}\) is linearly dependent.
    ðŸ”—
    ðŸ”—

    Example 8.2.3.

    Show that the set \(\{\vect{e}_1, \vect{e}_2\}\) is linearly independent in \(\R^2\text{.}\)
    ðŸ”—
    Suppose \(c_1\vect{e}_1 + c_2\vect{e}_2 = \vect{0}\text{.}\) Then
    \begin{equation*} c_1[1, 0] + c_2[0, 1] = [c_1, c_2] = \vect{0}, \end{equation*}
    so \(c_1 = 0\) and \(c_2 = 0\text{.}\) Therefore \(\{\vect{e}_1, \vect{e}_2\}\) is linearly independent.
    ðŸ”—
    ðŸ”—
    The next theorem shows how linear independence abstracts the idea of vectors not being parallel.
    ðŸ”—

    Theorem 8.2.4.

    A set of two vectors in \(\R^n\) is linearly independent if and only if neither vector is a scalar multiple of the other.
    ðŸ”—
    Equivalently, \(\{\vect{v}_1, \vect{v}_2\} \subset \R^2\) is linearly dependent if and only if at least one of the vectors is a scalar multiple of the other.
    ðŸ”—
    ðŸ”—

    Proof.

    Suppose that one of the vectors is a scalar multiple of the other; by relabelling the vectors if necessary, we can assume that
    \begin{equation*} \vect{v}_2 = c\vect{v}_1 \end{equation*}
    for \(c \in \R\text{.}\) Then \(c\vect{v}_1 - \vect{v}_2 = \vect{0}\text{.}\) Since the coefficient of \(\vect{v}_2\) is nonzero, the set \(\{\vect{v}_1, \vect{v}_2\}\) is linearly dependent.
    ðŸ”—
    Conversely, suppose that \(\{\vect{v}_1, \vect{v}_2\}\) is linearly dependent. Then there exists \(c_1, c_2 \in \R\text{,}\) not both zero, such that
    \begin{equation*} c_1\vect{v}_1 + c_2\vect{v}_2 = \vect{0}\text{.} \end{equation*}
    We can assume \(c_1\ne 0\text{.}\) Dividing by \(c_1\) gives \(\vect{v}_1 + \frac{c_2}{c_1}\vect{v}_2 = \vect{0}\text{,}\) so
    \begin{equation*} \vect{v}_1 = -\frac{c_2}{c_1}\vect{v}_2, \end{equation*}
    so \(\vect{v}_1\) is a scalar multiple of \(\vect{v}_2\text{.}\)
    ðŸ”—
    ðŸ”—
    For \(k\) vectors, we have a similar result. However, to show linear independence in this case it is not enough to check that none of the vectors are parallel. We also need to check that none of the vectors can be written as linear combination of the other vectors.
    ðŸ”—

    Theorem 8.2.5.

    A set of vectors \(\{\vect{v}_1, \vect{v}_2, \dots, \vect{v}_k\}\) in \(\R^n\) is linearly independent if and only if none of the vectors in the set can be expressed as a linear combination of the others.
    ðŸ”—
    ðŸ”—
    The proof is very similar to that of TheoremÂ 8.2.4; there is not much difference apart from replacing \(\vect{v}_1, \vect{v}_2\) with \(\vect{v}_1, \dots, \vect{v}_k\text{.}\)
    ðŸ”—
    We now come to the connection between linear independence and matrix inverses.
    ðŸ”—

    Theorem 8.2.6.

    Let
    \begin{equation*} A = \begin{bmatrix} \vect{a}_1 \amp \vect{a}_2 \amp \cdots \amp \vect{a}_n \end{bmatrix} \end{equation*}
    be an \(n\times n\) matrix with columns \(\vect{a}_1\text{,}\) \(\vect{a}_2\text{,}\) \(\dots, \vect{a}_n\text{.}\) Then \(A\) is invertible if and only if the set \(\{\vect{a}_1\text{,}\) \(\vect{a}_2\text{,}\) \(\dots, \vect{a}_n\}\) is linearly independent.
    ðŸ”—
    ðŸ”—

    Proof.

    Let \(\vect{x} \in \R^n\) be arbitrary. Write \(\vect{x} = [c_1, c_2, \dots, c_n]\) and \(A = [a_{ij}]\text{.}\) Then
    \begin{align*} A\vect{x} \amp = \begin{bmatrix} a_{11} \amp a_{12} \amp \cdots \amp a_{1n} \\ a_{21} \amp a_{22} \amp \cdots \amp a_{2n} \\ \vdots \amp \vdots \amp \ddots \amp \vdots \\ a_{n1} \amp a_{n2} \amp \cdots \amp a_{nn} \end{bmatrix} \begin{bmatrix} c_1 \\ c_2\\ \vdots\\ c_n \end{bmatrix} = \begin{bmatrix} c_1 a_{11} + c_2a_{12} + \dots + c_na_{1n} \\ c_1 a_{21} + c_2a_{22} + \dots + c_na_{2n} \\ \vdots \\ c_1 a_{n1} + c_2a_{n2} + \dots + c_na_{nn} \end{bmatrix} \\ \amp = c_1 \begin{bmatrix} a_{11} \\ a_{21}\\ \vdots\\ a_{n1} \end{bmatrix} + c_2 \begin{bmatrix} a_{12} \\ a_{22}\\ \vdots\\ a_{n2} \end{bmatrix} + \dots + c_n \begin{bmatrix} a_{1n} \\ a_{2n}\\ \vdots\\ a_{nn} \end{bmatrix} \\ \amp = c_1\vect{a}_1 + c_2\vect{a}_2 + \dots + c_n\vect{a}_n\text{.} \end{align*}
    From TheoremÂ 7.6.1, \(A\) is invertible if and only if \(A\vect{x} = \vect{0}\) has the unique solution \(\vect{x} = \vect{0}\text{.}\) Using the previous calculation and that \(\vect{x} = [c_1, c_2, \dots, c_n]\text{,}\) this shows that \(A\) is invertible if and only if
    \begin{equation*} c_1\vect{a}_1 + c_2\vect{a}_2 + \dots + c_n\vect{a}_n = \vect{0} \end{equation*}
    has the unique solution \(c_1 = c_2 = \dots = c_n = 0\text{,}\) which is the definition of \(\{\vect{a}_1, \vect{a}_2, \dots, \vect{a}_n\}\) being linearly independent.
    ðŸ”—
    ðŸ”—

    Example 8.2.7.

    Consider the matrix
    \begin{equation*} A = \begin{bmatrix} 1 \amp 1 \\ 2 \amp 3 \end{bmatrix} \end{equation*}
    Its columns \([1, 2]\) and \([1, 3]\) are not parallel and hence are linearly independent, so \(A\) is invertible. This agrees with the calculation that \(\det(A) = 3 - 2 = 1 \ne 0\text{.}\)
    ðŸ”—
    ðŸ”—
    For larger matrices, it is not easy to tell whether all the columns are linearly independent. But if they are linearly dependent, sometimes this is quick to spot, which can allow us to quickly conclude that the matrix is not invertible.
    ðŸ”—

    Example 8.2.8.

    The matrix
    \begin{equation*} A = \begin{bmatrix} \vect{a}_1 \amp \vect{a}_2 \amp \vect{a}_3 \amp \vect{a}_4 \end{bmatrix} = \begin{bmatrix} 4 \amp 9 \amp 8 \amp 6 \\ 1 \amp 8 \amp 2 \amp 2 \\ 0 \amp 1 \amp 0 \amp 5 \\ 4 \amp 2 \amp 8 \amp 3 \end{bmatrix} \end{equation*}
    has \(\vect{a}_3 = 2\vect{a}_1\text{,}\) so its columns are linearly dependent. Therefore \(A\) is not invertible.
