    Section 1.3 Vectors in \(\mathbb{R}^n\)

    Vectors in \(\R^n\) are very similar to vectors in the plane, except they have \(n\) components instead of 2.
    ðŸ”—

    Definition 1.3.1. Vectors in \(\R^n\).

    A vector \(\vect{a}\) in \(\R^n\) is an expression of the form
    \begin{equation*} \vect{a} = \begin{bmatrix} a_1 \\ a_2\\ \vdots\\ a_n \end{bmatrix} , \end{equation*}
    or \(\vect{a} = [a_1, a_2, \dots, a_n]\) to save space, with components \(a_1, \dots, a_n \in \R\text{.}\)
    ðŸ”—
    ðŸ”—
    The definition of vector addition and scaling in \(\R^n\) is component-wise, like in \(\R^2\text{.}\) Let \(c\in \R\) be a scalar, and \(\vect{a} = [a_1, a_2, \dots, a_n], \vect{b} = [b_1, b_2, \dots, b_n] \in \R^n\) be vectors. Then we define the sum
    \begin{equation*} \vect{a} + \vect{b} = [a_1 + b_1, a_2 + b_2, \dots, a_n+b_n] \end{equation*}
    and the scalar multiple
    \begin{equation*} c\vect{a} = [ca_1, ca_2, \dots, ca_n]\text{.} \end{equation*}
    ðŸ”—
    The zero vector in \(\R^n\) is the vector
    \begin{equation*} \vect{0} = [0, 0, \dots, 0] \end{equation*}
    having \(n\) components which are all zero.
    ðŸ”—
    A vector in \(\R^n\) can still be interpreted as a directed line segment in \(n\)-dimensional space. In \(\R^3\text{,}\) we can still draw vectors, add vectors tip to tail, and interpret scaling as scaling in \(3\)-dimensional space. See FigureÂ 1.3.2 for how to plot a vector in \(\R^3\) using its components. In \(\R^n\) for \(n\ge 4\text{,}\) we canâ€™t draw vectors. However, our algebraic methods developed for \(\R^2\) still work for \(\R^n\text{,}\) without any issues.
    ðŸ”—

    Remark 1.3.3.

    We can view vectors in \(\R^1\) as real numbers: for \([t] \in \R^1\text{,}\) we consider this the same as \(t \in \R\text{.}\) For example, we view the vector \([2]\) as the number 2.
    ðŸ”—
    ðŸ”—

    Example 1.3.4.

    In \(\R^5\text{,}\) let \(\vect{a} = [1, 0, 2, 3, 1]\) and \(\vect{b} = [2, 1, 0, 1, 4]\text{.}\) Then
    \begin{align*} \vect{a} + \vect{b} \amp = [1+2, 0+1, 2+0, 3+1, 1+4] \\ \amp = [3, 1, 2, 4, 5], \\ \vect{a} - \vect{b} \amp = [1-2, 0-1, 2-0, 3-1, 1-4] \\ \amp = [-1, -1, 2, 2, -3], \\ 2\vect{a} \amp = [2, 0, 4, 6, 2]. \end{align*}
    ðŸ”—
    ðŸ”—
    The following fundamental properties of vectors in \(\R^n\) describes how addition, scaling, negation, and the zero vector interact. We will call this a `Theoremâ€™, which is a name in mathematics for a fact or result, usually an important one.
    ðŸ”—

    Theorem 1.3.5.

    Let \(\vect{u}, \vect{v}, \vect{w} \in \R^n\) and \(c, d \in \R\text{.}\) Then
    1. \(\vect{u} + \vect{v} = \vect{v} + \vect{u}\quad\) (commutativity)
      ðŸ”—
      ðŸ”—
    2. \((\vect{u} + \vect{v}) + \vect{w} = \vect{u} + (\vect{v}+ \vect{w})\quad\) (associativity)
      ðŸ”—
      ðŸ”—
    3. \(\displaystyle \vect{u} + \vect{0} = \vect{u}\)
      ðŸ”—
      ðŸ”—
    4. \(\displaystyle \vect{u} + (-\vect{u}) = \vect{0}\)
      ðŸ”—
      ðŸ”—
    5. \(c(\vect{u} + \vect{v}) = c\vect{u} + c\vect{v}\quad\) (distributivity)
      ðŸ”—
      ðŸ”—
    6. \((c+d)\vect{u} = c\vect{u} + d\vect{u}\quad\) (distributivity)
      ðŸ”—
      ðŸ”—
    7. \(\displaystyle c(d\vect{u}) = (cd)\vect{u}\)
      ðŸ”—
      ðŸ”—
    8. \(1\vect{u} = \vect{u}\text{.}\)
      ðŸ”—
      ðŸ”—
    ðŸ”—
    ðŸ”—
    Properties 5 and 6 are both known as distributivity. Associativity (Property 2) allows us to write sums of several vectors
    \begin{equation*} \vect{u} + \vect{v} + \vect{w} \end{equation*}
    without using any brackets, because no matter what order the sums are performed in, the resulting vector will be same.
    ðŸ”—
    Theorems are closely related to `proofsâ€™, which is a mathematical demonstration of why the fact is true. We will prove Properties 3-6 to give an indication of how to do it. The proof of the remaining properties will be left as an exercise.
    ðŸ”—

    Proof.

    Let \(\vect{u} = [u_1, u_2, \dots, u_n]\) and \(\vect{v} = [v_1, v_2, \dots, v_n]\text{.}\) Then Property 3 is proved by
    \begin{align*} \vect{u} + \vect{0} \amp = [u_1, u_2, \dots, u_n] + [0, 0, \dots, 0] \\ \amp = [u_1 + 0, u_2 + 0, \dots, u_n + 0] \\ \amp = \vect{u}\text{,} \end{align*}
    Property 4 holds since
    \begin{equation*} \vect{u} + (-\vect{u}) = [u_1 - u_1, \dots, u_n - u_n] = \vect{0}, \end{equation*}
    and Property 5 holds since
    \begin{align*} c(\vect{u} + \vect{v}) \amp = c([u_1 + v_1, u_2 + v_2, \dots, u_n + v_n]) \\ \amp = [cu_1 +cv_1, cu_2 + cv_2, \dots, cu_n + cv_n] \\ \amp = c\vect{u} + c\vect{v}\text{.} \end{align*}
    ðŸ”—
    Finally, the following calculation
    \begin{align*} (c+d)\vect{u} \amp = (c+d)[u_1, u_2, \dots, u_n] \\ \amp = [(c+d)u_1, (c+d)u_2, \dots, (c+d)u_n] \\ \amp = [cu_1 + du_1, cu_2 + du_2, \dots, cu_n + du_n] \\ \amp = c\vect{u} + d\vect{u}\text{.} \end{align*}
    proves Property 6.
    ðŸ”—
    ðŸ”—
    We can use these properties to simplify vector expressions.
    ðŸ”—

    Example 1.3.6.

    Let \(\vect{a}, \vect{b} \in \R^n\text{.}\) The expression \(3(2\vect{a} - \vect{b}) + (\vect{a} - 3\vect{b})\) is simplified by
    \begin{align*} \amp 3(2\vect{a} - \vect{b}) + (\vect{a} - 3\vect{b}) \\ \amp\quad = 3(2\vect{a} + (-1)\vect{b}) + (\vect{a}+(-1)3\vect{b}) \quad\text{(definition of subtraction)} \\ \amp\quad = [3(2\vect{a}) + 3(-1\vect{b})] + (\vect{a}+(-1)3\vect{b}) \quad\text{(distributivity)} \\ \amp\quad =[6\vect{a}+(-3\vect{b})] + (\vect{a} + (-3)\vect{b}) \quad\text{(Property 7)} \\ \amp\quad = 6\vect{a} + \vect{a} + (-3)\vect{b} + (-3)\vect{b} \quad\text{(associativity and commutativity)} \\ \amp\quad = (6+1)\vect{a} + ((-3)+(-3))\vect{b} \quad\text{(Property 8, distributivity)} \\ \amp\quad = 7\vect{a} + (-6)\vect{b} \quad\text{(addition in }\R\text{)} \\ \amp\quad = 7\vect{a} - 6\vect{b}. \quad\text{(Property 7, definition of subtraction)} \end{align*}
    ðŸ”—
    ðŸ”—
    We went through this example in detail to show how all the properties are applied. But in practice TheoremÂ 1.3.5 means that we can simplify vector algebra intuitively, using similar rules to algebra with numbers. So we can combine most of the steps to do it faster, for example
    \begin{equation*} 3(2\vect{a} - \vect{b}) + (\vect{a} - 3\vect{b}) = 6\vect{a} - 3\vect{b} + \vect{a} - 3\vect{b} = 7\vect{a} - 6 \vect{b}\text{.} \end{equation*}
    ðŸ”—
    So far, we know how to add and scale vectors. We will give a name to what happens when we combine addition with scaling.
    ðŸ”—

    Definition 1.3.7.

    A vector \(\vect{v}\in \R^n\) is a linear combination of vectors \(\vect{v}_1, \vect{v}_2, \dots, \vect{v}_k\) if there are scalars \(c_1, c_2, \dots, c_k \in \R\) such that
    \begin{equation*} \vect{v} = c_1\vect{v}_1 + c_2\vect{v}_2 + \dots + c_k\vect{v}_k\text{.} \end{equation*}
    ðŸ”—
    The scalars \(c_1, c_2, \dots, c_k\) are called the coefficients of the linear combination.
    ðŸ”—
    ðŸ”—

    Example 1.3.8.

    Write \(\vect{u} = [2, 2]\) as a linear combination of \(\vect{v} = [-1, 1]\) and \(\vect{w} = [0, 4]\text{.}\) Let \(\vect{u} = c_1\vect{v} + c_2\vect{w}\text{,}\) i.e\
    \begin{equation*} [2, 2] = c_1[-1, 1] + c_2[0, 4]\text{.} \end{equation*}
    ðŸ”—
    Comparing components gives
    \begin{equation*} \begin{aligned} 2 \amp = -c_1 \\ 2 \amp = c_1 + 4c_2. \\ \end{aligned} \end{equation*}
    so \(c_1 = -2, c_2 = \frac{1}{4}(2-c_1) = 1\text{.}\) Thus \(\vect{u} = -2\vect{v} + \vect{w}\text{.}\) See FigureÂ 1.3.9.
    ðŸ”—
    ðŸ”—

    Example 1.3.10.

    Is it possible to write \(\vect{u} = [2, 2]\) as a linear combination of \(\vect{v} = [-1, 1]\) and \(\vect{x} = [3, -3]\text{?}\)
    ðŸ”—
    Suppose we try to write \(\vect{u} = c_1\vect{v} + c_2\vect{x}\text{,}\) so
    \begin{equation*} [2, 2] = c_1[-1, 1] + c_2[3, -3]\text{.} \end{equation*}
    ðŸ”—
    Comparing coefficients gives
    \begin{equation*} \begin{aligned} 2 \amp = -c_1 + 3c_2 \\ 2 \amp = c_1 - 3c_2. \\ \end{aligned} \end{equation*}
    ðŸ”—
    Adding these equations gives \(4=0\text{,}\) which is a contradiction. Therefore \(\vect{u}\) cannot be written as a linear combination of \(\vect{v}\) and \(\vect{x}\text{.}\) This is also shown in FigureÂ 1.3.11: since \(\vect{u}\) does not lie on the dashed line, it cannot be a linear combination of \(\vect{v}\) and \(\vect{x}\text{.}\)
    ðŸ”—
    ðŸ”—
    In both of these examples, we had to solve a system of linear equations. This can be done by scaling and elimination (subtraction). However, in ChapterÂ 5, we will introduce augmented matrices which provide a better way of solving systems of linear equations.
